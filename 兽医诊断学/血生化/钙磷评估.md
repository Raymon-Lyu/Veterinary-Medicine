# 钙指标解读

## 基础认识
 - 参与钙代谢的器官：骨骼、肾脏、小肠
##### 参与改调节的激素
1. **PTH（甲状旁腺激素）**
    - ↑ Ca
    - ↓ Phos
2. **Vit D**
    - ↑ Ca
    - ↑ Phos
3. **降钙素**
    - ↓ Ca
    - ↓ Phos
## 高钙血症
### 诱发病因
一个神奇的口诀：
> **Hard ions GG**

对应为：
- **H** = Hyperparathyroidism（甲状旁腺功能亢进）
- **A** = Addisons（阿狄森氏病）
- **R** = Renal disease（肾脏疾病）
- **D** = Vitamin D toxicosis（Vit D 中毒）
- **I** = Idiopathic（特发性）
- **O** = Osteolysis（骨溶解）
- **N** = Neoplasia（肿瘤）
- **S** = Spurious（假性）
- **G** = granulomatous（肉芽肿性）
- **G** = growth（生长期）
##### 不同物种常见病因
- 犬：肿瘤、甲状旁腺功能亢进、肾脏疾病
- 猫：特发性、肿瘤、慢性肾病（CKD）

### 恶性高钙血症
- 常见相关肿瘤
	1. 淋巴瘤
	2. 顶泌腺肛门囊腺癌（AGASACA）
	3. 多发性骨髓瘤
- 其他相关肿瘤
	- 甲状腺癌
	- 胸腺瘤
	- 肺癌
	- 乳腺癌
	- 腺癌
	- 鳞癌
	- 甲状旁腺肿瘤
	- 黑色素瘤
	- 骨肿瘤
	- 慢性淋巴细胞白血病
	- 肾细胞癌
	- 鼻癌
### 临床表现
- **心脏**：心律不齐
- **胃肠道**：厌食、呕吐、便秘
- **神经**：精神沉郁、虚弱、癫痫、肌肉痉挛
- **肾脏**：多饮多尿、氮质血症、肾小管矿化、尿路结石
### 判读逻辑
发现高钙血症时，优先问：
1. **是不是肿瘤相关？**
	- 肿瘤分泌PTH类似物(PTHrP)
	- 肿瘤侵袭骨或骨转移
	- 淋巴瘤通过升高calcitriol(异常活性$V_D$)
2. **是不是甲状旁腺功能亢进？**
3. **是不是肾病？**
4. **是不是阿狄森？**
5. **是不是 Vit D 中毒？**
6. **是不是假性高钙？**
	- **总钙高 + 离子钙正常** → 假性高钙
	- 同时有**脱水、白蛋白升高、总蛋白升高、球蛋白升高**

## 低钙血症
### 诱发病因
- [[蛋白质评估#低白蛋白血症|低白蛋白血症]]

> 所以低钙时一定要先想：  
> **这是总钙低，还是离子钙真的低？**

##### 生理/围产期相关
- 怀孕、泌乳
- 牛：临产低钙
- 犬、马、羊：产后抽搐 / 子痫惊厥
##### 甲状旁腺/维生素D异常
- 原发性甲状旁腺功能减退
- 继发性营养性甲状旁腺功能亢进
- 慢性肾病继发性甲旁亢
- 维生素 D 缺乏
##### 其他疾病或代谢紊乱
- 外分泌胰功能不全
- 蛋白丢失性肠病
- 高降钙素血症
- 草酸盐中毒
- 乙二醇中毒
- 代谢性碱中毒
- 速尿治疗
- 急性胰腺炎
- 尿路梗阻
- 急性肾损伤
- 磷灌肠剂
- EDTA / 柠檬酸盐样本污染
- 马腹绞痛
- 斑蝥中毒
- 肌病
- 急性肿瘤溶解综合征
- 马大量出汗
- 吃粮过量
- 肾上腺皮质机能亢进
- 低镁血症

### 判读逻辑
低钙时按以下顺序想：
1. **先排低白蛋白**
2. **再想围产期低钙**
3. **再想甲状旁腺 / Vit D 轴异常**
4. **再想急性疾病、毒物、样本问题**
## 总结
### 高钙血症
- 犬：**肿瘤 > 甲旁亢 > 肾病**
- 猫：**特发性 > 肿瘤 > CKD**
### 低钙血症
- **先看白蛋白**
- 再想：
    - 产科相关
    - 甲旁腺相关
    - Vit D 相关
    - 胰腺炎/AKI/尿路梗阻/毒物

# 磷指标解读
## 高磷血症
### 病因机制
##### 尿液排泄减少
- 肾脏疾病（GFR↓）
- 膀胱破裂
- 甲状旁腺功能减退
- 肢端肥大症（生长激素↑ → 肾小管重吸收↑）
##### 肠道吸收增加
- 外源性磷酸盐
- Vit D 增多
- 饮食 Ca:P 比下降
##### 细胞内液转向细胞外液
- 肌病
- 急性肿瘤溶解综合征
##### 其他
- 甲状腺机能亢进
- 肾上腺皮质功能亢进
- 乳酸中毒
##### 假性高磷血症
- 高胆红素血症
- 单克隆丙球蛋白病
- 体外溶血

### 判读逻辑
高磷血症最先考虑：
1. **肾排不出去**
    - CKD / AKI / 膀胱破裂
2. **甲旁腺相关调节异常**
3. **细胞破坏释放**
    - 肌损伤
    - 肿瘤溶解
4. **是假象吗？**
    - 溶血
    - 高胆红素
    - 单克隆球蛋白异常
##  低磷血症
### 病因机制
#####  尿液排泄增加
- prolonged diuresis（长期多尿）
- 范可尼综合征
##### 肠道吸收减少
- 长期厌食
- 低磷饮食
- Vit D下降
- 吸收不良
##### 细胞外液转入细胞内液
- 高胰岛素血症
- 葡萄糖输注
- 呼吸性碱中毒
##### 从骨骼动员不足
- 奶牛热
- 子痫
- 马肾脏疾病
- 马氟烷麻醉

### 判读逻辑
低磷血症重点考虑：
1. **吃得少 / 吸收差**
2. **排得多**
3. **进细胞里去了**
4. **围产期或骨代谢动员异常**
# 钙磷联合判读
## 先看钙，再看磷

因为 Ca 和 Phos 受到共同调控，但异常组合不同，提示方向也不同：
### 高 Ca + 低/正常 Phos
优先考虑：
- 甲状旁腺功能亢进  
    因为 **PTH：↑Ca、↓Phos**
### 高 Ca + 高 Phos
优先考虑：
- Vit D 中毒
- 肾脏疾病  
    因为：
- Vit D：**↑Ca、↑Phos**
- 肾病会影响磷排泄
### 低 Ca + 高 Phos
优先考虑：
- 甲状旁腺功能减退
- AKI / 尿路梗阻
- 磷灌肠剂等外源性负荷
### 低 Ca + 低 Phos
优先考虑：
- 围产期低钙
- 奶牛热
- 子痫
- 营养或吸收问题

## 总结表

| 组合               | 优先考虑          |
| ---------------- | ------------- |
| 高 Ca + 低/正常 Phos | 甲旁亢           |
| 高 Ca + 高 Phos    | Vit D 中毒、肾病   |
| 低 Ca + 高 Phos    | 甲旁减退、AKI、尿路梗阻 |
| 低 Ca + 低 Phos    | 围产期低钙、营养/吸收问题 |
