# 心脏的检查
心脏的检查流程有如下结构：
```mermaid
flowchart LR
A[心脏检查] --> B[心搏动检查]
B --> G[指标<br>强度、频率、敏感型]
A --> C[心脏叩诊]
C --> H[叩诊区的确定]
C --> I[叩诊音的变化]
A --> D[心脏听诊]
D --> E[生理性心音]
E --> K[第一心音<br>第二心音]
D --> F[病理性心音]
F --> J[心音增强/减弱、性质改变、心音分裂、节律改变、心杂音]
```

进行心脏的诊断，首先要确定心脏的位置：
- 胸腔的下$\frac{1}{3}$处
- 位于第三肋到第六肋之间
- 胸腔正中线的左侧
- 基底部向右前，尖部向左后

## 心搏动检查
**心搏动**：心室收缩撞击左侧胸壁形成的碰撞音，与第一心音、脉搏同时出现
### 频率
常见动物的心搏动频率
- 🐕：70-120 bpm
- 🐖：60-80 bpm
- 🐎：26-42 bpm
- 🐏：70-80 bpm
### 强度
病理性变化有三种：
1. 增强：心室收缩力增强，见于热性病、贫血、心脏疾病代偿
> **心悸**：心搏动过强引起**体壁**的震动
2. 减弱：心脏机能减弱或远离胸壁，见于心力衰竭、胸壁增厚(水肿、纤维素性胸膜炎)、胸壁心脏介质性质改变
3. 移位：心脏受周围脏器或肿瘤及其渗出液压迫产生位移
### 敏感性
提高见于心包炎、胸膜炎

## 心脏叩诊
### 叩诊区确定
- 绝对浊音区：心脏靠近胸壁的部分，与胸壁直接接触
- 相对浊音区：心脏的主体，被肺脏包裹
### 叩诊音变化
1. 浊音区增大：见于心肥大、心扩张、心包炎
2. 浊音区减小：见于肺气肿
3. 叩诊音呈[[呼吸系统临床检查#鼓音|鼓音]]或[[呼吸系统临床检查#金属音|金属音]]：见于牛创伤性心包炎
4. 叩诊带痛：见于胸膜炎、心包炎

## 心脏听诊
### 正常心音
*心音形成机制可以结合心肌活动+瓣膜运动+血流及血柱振动来记忆*
- 第一心音：
	- 缩期心音
	- 心室肌收缩、二/三尖瓣关闭、半月瓣开放
- 第二心音：
	- 舒期心音
	- 心室肌弛缓、二/三尖瓣开放、半月瓣关闭

| 比较维度 | 第一心音（S1） | 第二心音（S2） |
|---|---|---|
| 产生时期 | 心室收缩开始（收缩期） | 心室舒张开始（舒张期） |
| 形成机制 | 房室瓣关闭+ 心室肌张力变化 + 血流振动 | 主动脉瓣和肺动脉瓣关闭 |
| 持续时间 | 较长 | 较短 |
| 音调 | 较低 | 较高、较清脆 |
| 强度特点 | 较钝厚 | 较清亮 |
| 尾音 | 有较长尾音 | 无明显尾音 |
| 与心搏动关系 | 与心尖搏动同时出现 | 不与心尖搏动同步 |
| 与动脉脉搏关系 | 与动脉脉搏几乎同时 | 晚于动脉脉搏出现 |
| 两心音间隔 | S1→S2 间隔短 | S2→下一次 S1 间隔长 |

##### 心音听取位点
心音听取位点 = 各瓣膜音或相关血流音在胸壁上的最佳传导/最佳听诊区域
如：马的第一心音的二尖瓣听取位点在左侧第**5**肋间，胸廓下$\frac{1}{3}$的中央水平线上
### 病理变化
#### 心音增强
1. 第一心音增强：心脏收缩力增强+心室充盈度小
	- 心肌收缩增强：房室瓣关闭更突然，振动幅度更大
	- 心室充盈不足：见于大失血/水、休克，房室瓣的开放程度更大
	- 血液稀薄、流速快
2. 第二心音增强：循环压增高
	- 主动脉瓣：主动脉压高，见于左心肥大、肾炎、系统性高血压
	- 肺动脉瓣：肺动脉压高，见于肺充血、肺炎、二尖瓣闭锁不全
3. 两者均增强：心脏整体搏动增强
	- 交感兴奋/循环加快：热性病初期、剧痛、兴奋、[[血液循环系统药理#强心药|强心药的使用]]
	- 高动力循环：贫血/心脏病代偿
	- 肺组织萎缩
#### 心音减弱
1. S1减弱：心肌收缩无力+房室瓣关闭振动减弱
	见于心扩张、心肌变性
2. S2减弱：动脉压低+半月瓣关闭受限
	见于主(肺)动脉瓣的闭锁不全、狭窄
3. 两者均减弱：整体性的减弱
	- 心肌收缩力减弱：心脏自身功能受限，如心功能不全、心肌变性/梗死
	- 心音传导介质改变：液体/渗出物/组织增厚削弱了心音的传导，见于渗出性心包炎、渗出性胸膜炎、心包积水
	- 胸壁/肺对声音的阻挡$\uparrow$：胸壁增厚、肺气肿、胸膜/心包粘连
#### 心音性质变化
- 心音浑浊：心音低浊、含混不清，S1和S2之间界限难以区分
	见于心肌变性、心肌炎、心内膜炎
- 金属样心音：类比于[[呼吸系统临床检查#金属音|金属音]]，心脏临近肺脏出现空洞或破伤风
#### 心音分裂
- **定义**：正常心音由于某些病理原因而分裂成两个相连的心音，分裂出的心音间隔称为**心音重复**
心音分裂有三种常见情况：
1. 第一心音分裂：二/三尖瓣未同时关闭，见于心室一侧房室束传导受限或心肌变性
2. 第二心音分裂：主动脉瓣和肺动脉瓣的关闭不同时
	- 两动脉压力差明显：压力大的一侧延迟关闭
	- 两心室充盈程度不同：充盈量大射血期长延迟关闭
3. 奔马律：在正常心音外产生的额外心音
	- 缩期前奔马调：房室束传导受限导致心房音被放大，见于心肌炎、主(肺)动脉口狭窄
	- 舒张初期奔马调：血液快速涌入+心室壁顺应性差，见于心力衰竭、心肌损伤

> - S3：**快充盈**、**容量负荷**、**扩张心室**、**振动增强**
> - S4：**硬心室**、**顺应性差**、**心房收缩硬推进去**


#### 心音律动周期改变
正常动物心音出现的强弱、间隔一致，出现改变成为**心律不齐**
- 期外收缩：窦房结外的起搏点引发搏动，带来代偿性间歇
	- 存在代偿间歇期
	- 脉搏短缺：射血量低
	- S1增强，S2减弱或消失
- 阵发性心动过速：出现多次($\geq 3$)期外收缩的心率周期变化
- 心动间歇：几次正常心动周期后出现一次停跳，多由于心肌的传导功能障碍
#### 心杂音
- **定义**：除正常心音外因心脏搏动而产生的附加音
```mermaid
mindmap
  root((心杂音))
    心内性杂音
      器质性
        瓣膜闭锁不全
        瓣孔狭窄
      功能性
        相对闭锁不全
        贫血性杂音
    心外性杂音
      心包性
        心包摩擦音
        心包击水音
      心包外
        心包胸膜摩擦音
```

##### 心内杂音
- 由心内瓣膜、瓣口或血液动力学改变引起的杂音
###### 器质性
- 由瓣膜或瓣口的形态学改变引起的杂音
- 机制：瓣膜的器质性改变$\to$血流紊乱，涡流产生$\to$振动心壁产生杂音
- 收缩期杂音提示房室瓣的闭锁不全或动脉口狭窄；舒张期杂音提示房室口狭窄或动脉瓣闭锁不全
> 具体对应关系解析可见[[心内杂音机制]]
- 具有以下特点：
	- 持续性强，难逆转
	- 心功能增强明显
	- 音色尖锐，呈灌注音、流水音、笛音
###### 功能性
- 瓣膜本身无变化，但是其功能或血流性质发生了改变
由于其功能的定义，故可以继续分为两个子类型：
1. 相对闭锁不全杂音
	瓣膜本身正常，但心腔扩大或支持结构失调，如心室扩张、心脏高度弛缓、乳头肌功能障碍
2. 贫血性杂音
	见于重度贫血或血液稀薄流速加快时，冲击瓣膜等结构产生杂音
- 具有以下特点：
	- 音色较柔和，不稳定、波动大
	- 运动、强心或随病情恢复而减弱
	- 仅见于**收缩期**
##### 心外杂音
- 由心包或邻近心区胸膜病引起的杂音
主要由以下几种类型：
1. 心包摩擦音：提示纤维素性心包炎
2. 心包拍水音：提示心包积水
3. 心包胸膜摩擦音：提示纤维素性胸膜炎

##### 总结
心内性与心外性总结的如下：

|项目|心内性杂音|心外性杂音|
|---|---|---|
|来源|瓣膜、瓣口、异常血流|心包或邻近胸膜|
|本质|血流涡流 + 内部振动|摩擦或液体震荡|
|常见类型|闭锁不全、狭窄、贫血性|心包摩擦音、拍水音|
|与呼吸关系|一般不明显|可受呼吸影响|
|诊断意义|侧重瓣膜病、血流动力学异常|侧重心包炎、胸膜炎|
