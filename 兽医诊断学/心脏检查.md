# 心脏的检查
心脏的检查流程有如下结构：
```mermaid
flowchart LR
A[心脏检查] --> B[心搏动检查]
B --> G[指标<br>强度、频率、敏感型]
A --> C[心脏叩诊]
C --> H[叩诊区的确定]
C --> I[叩诊音的变化]
A --> D[心脏听诊]
D --> E[生理性心音]
E --> K[第一心音<br>第二心音]
D --> F[病理性心音]
F --> J[心音增强/减弱、性质改变、心音分裂、节律改变、心杂音]
```

进行心脏的诊断，首先要确定心脏的位置：
- 胸腔的下$\frac{1}{3}$处
- 位于第三肋到第六肋之间
- 胸腔正中线的左侧
- 基底部向右前，尖部向左后

## 心搏动检查
**心搏动**：心室收缩撞击左侧胸壁形成的碰撞音，与第一心音、脉搏同时出现
### 频率
常见动物的心搏动频率
- 🐕：70-120 bpm
- 🐖：60-80 bpm
- 🐎：26-42 bpm
- 🐏：70-80 bpm
