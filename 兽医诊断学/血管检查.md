# 血管检查
## 动脉检查
主要考察动脉的脉数、脉性、脉律
### 脉搏的大小
- 脉搏的振幅高低能反映心脏的收缩力、搏出血量、血管紧张度
具体可以分为：

| 项目        | 大脉                        | 小脉                        |
| --------- | ------------------------- | ------------------------- |
| **定义**    | 脉搏振幅增高                    | 脉搏振幅降低                    |
| **形成机制**  | 强收缩 + 大搏出量 + 血管松弛 → 压力波明显 | 弱收缩 + 小搏出量 + 血管紧张 → 压力波减弱 |
| **临床提示**  | 循环高动力状态                   | 低灌注 / 心衰状态                |
| **常见疾病**  | 发热初期、左心肥大、心脏代偿期、主动脉瓣闭锁不全  | 大失血、大失水、心力衰竭、主动脉瓣口狭窄      |
### 紧张度
- 对血管施加压力时血管产生
