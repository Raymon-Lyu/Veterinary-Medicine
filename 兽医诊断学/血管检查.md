# 血管检查
## 动脉检查
主要考察动脉的脉数、脉性、脉律
### 脉搏的大小
- 脉搏的振幅高低能反映心脏的收缩力、搏出血量、血管紧张度
具体可以分为：

| 项目       | 大脉                                                  | 小脉                                          |
| -------- | --------------------------------------------------- | ------------------------------------------- |
| **定义**   | 脉搏振幅增高                                              | 脉搏振幅降低                                      |
| **形成机制** | 强收缩 + 大搏出量 + 血管松弛 → 压力波明显                           | 弱收缩 + 小搏出量 + 血管紧张 → 压力波减弱                   |
| **临床提示** | 循环高动力状态                                             | 低灌注 / 心衰状态                                  |
| **常见疾病** | 发热初期、左心肥大、心脏代偿期、主动脉瓣闭锁不全(能射出去，而且射得多；但舒张期又漏回去，所以脉压大) | 大失血、大失水、心力衰竭、主动脉瓣口狭窄(射出去本身就困难，每搏输出小，所以脉搏弱小) |
### 紧张度
- 对血管施加压力时血管产生的抵抗力，反映出血管本身的张力、血压
具体可以分为：

|项目|硬脉|软脉|
|---|---|---|
|**定义**|指压血管时阻力增强|血管壁松弛，指压易消失|
|**核心本质**|**血管紧张度增高**|**血管紧张度下降**|
|**触诊感觉**|血管较“顶手”、不易压瘪|血管柔软、容易压瘪|
|**血管状态**|血管壁收缩、张力高|血管壁弛缓、张力低|
|**循环意义**|外周阻力升高|血压支撑能力下降|
|**常见疾病**|破伤风、肾炎、剧痛、铅中毒|心脏衰弱、大失血、脱水、恶病质、营养不良|
|**特殊类型**|**金线脉**：细小而极硬|**丝状脉**：细小而极软|

### 充盈度
- 通俗的讲，充盈度=动脉“鼓起来”的程度

|项目|实脉（满脉）|虚脉|
|---|---|---|
|**定义**|血管充盈良好，触之饱满|血管充盈不足，触之空虚|
|**核心本质**|**血量较多，充盈度高**|**血量较少，充盈度低**|
|**血管状态**|管腔较充实、较鼓满|管腔较空瘪、较不充实|
|**触诊感觉**|脉管饱满，触之较实|脉管空虚，触之不足|
|**决定因素**|血容量较足，血管充盈好|血容量不足或有效循环血量下降|
|**循环意义**|循环较充实，高动力或充血状态|循环不足，低血容量或心功能不全状态|
|**常见情况**|发热初期、心肥大、运动或使役后|大失血、大失水、心功能不全、主动脉瓣闭锁不全、恶病质、营养不良|
### 脉波形状
- 反应出动脉压力上升和下降的速度
具体可见：

|项目|迟脉|速脉（跳脉）|
|---|---|---|
|**定义**|脉波上升缓慢且持续，来得慢去得慢|脉波急剧上升又迅速下降，来得快去得快|
|**核心本质**|**射血进入动脉速度慢**|**射血迅猛 + 回落快**|
|**触诊感觉**|徐来而慢去|急来而骤去|
|**脉波特点**|波峰形成慢，下降也延长|波峰尖锐，下降迅速|
|**决定因素**|左心室射血受阻，主动脉压力上升慢|收缩期射血量大，舒张期压力快速下降|
|**典型机制**|**出口狭窄型**|**反流型 / 高搏出型**|
|**典型疾病**|主动脉口狭窄|主动脉瓣闭锁不全|
### 脉律
- 指脉搏搏动强度及间隙的均衡性，一致称为节律脉；不齐成为脉率不齐
- 脉率不齐是[[心脏检查#心音律动周期改变|心率不齐]]的结果
- 见于各种心脏病、疝痛、黄疸、颅压增加
## 静脉检查
静脉检查关注静脉淤血和颈静脉的波动
### 静脉淤血
- **定义**：静脉内血液回流受阻，导致静脉充盈、扩张、隆起。
- 机制：心脏功能减弱→静脉回流受阻  →全身静脉压升高→静脉充盈、怒张
- 临床表现
	- 颈静脉充盈、隆起
	- 静脉粗大如绳索
	- 可视黏膜树枝状充血
	- 常伴**低位性、对称性水肿**（心性水肿）
- 多提示**心力衰竭**尤其是**右心功能不全**
### 颈静脉波动
- 随心脏活动而出现的、由颈静脉基部向上部传播的逆行性波动，称为**颈静脉波动**。
主要分为三种类型：
> 一句话总结：阴性是生理回波，阳性是三尖瓣反流，假性是动脉传导

| 项目         | 阴性颈静脉波动                   | 阳性颈静脉波动                 | 假性颈静脉搏动            |
| ---------- | ------------------------- | ----------------------- | ------------------ |
| **本质**     | 生理性或轻度异常静脉波动              | 病理性静脉反流波动               | 动脉搏动传导到静脉的假象       |
| **主要机制**   | 心房收缩时静脉回流短暂受阻             | 心室收缩时血液经三尖瓣反流入右心房，再传至静脉 | 邻近动脉跳动过强影响颈静脉      |
| **对应时期**   | 舒张期                       | 与心室收缩一致                 | 与动脉搏动一致            |
| **波动性质**   | 心房性波动                     | 反流性波动                   | 非真正静脉波动            |
| **典型临床意义** | 可见于正常，也可在右心淤血时增强          | **三尖瓣闭锁不全**的重要特征        | 易误认为静脉异常，实为动脉影响    |
| **波动范围**   | 一般不超过下 1/3；明显时可达中 1/3甚至更高 | 多更明显，可达中 1/3            | 视觉上可见搏动，但不符合静脉反流规律 |
| **压迫试验结论** | 近、远端都消失                   | 近端加强，远端消失               | 近、远端都不消失           |
