# 泌尿系统检查
## 排尿状况检查
- 排尿的神经中枢位于腰荐部脊髓
检查主要包括：
1. 排尿姿势
2. 排尿次数和数量
### 排尿障碍
##### 频尿
- 总尿量不变，排尿次数增多，导致一次尿量减少
- 机制：膀胱、尿道粘膜受到刺激，刺激源包括尿路的炎症、结石
- 脊髓炎和疝痛是反射性引起排尿频数增加
##### 多尿
- 肾脏分泌量增加导致尿量、次数增加
- 见于慢性肾病、糖尿病、库欣综合征、[[水盐代谢#利尿药|利尿药]]的使用
##### 少尿
- 排尿次数、尿量减少
- 见于急性肾病、脱水、发热、心力衰竭
##### 无尿
- 膀胱内无尿，提示肾脏机能受损，引起水肿、尿毒症
- 见于肾功能衰竭、中毒、结石、肿瘤
##### 尿闭
- 亦称尿滞留，即肾脏产尿能力正常，膀胱排尿功能障碍
- 见于尿路阻塞、膀胱疾病
##### 尿淋漓
- 排尿不畅，表现为尿液呈点滴、细线或断续状排出
- 见于膀胱、尿道的炎症
- 疼痛性的尿淋漓称为**膀胱里急后重**
##### 尿失禁
- 无排尿动作但伴有尿液排出
- 见于脊髓的损伤、膀胱括约肌的麻痹
##### 排尿痛苦
- 排尿过程伴有动物痛苦表现
## 尿液状况检查
### 气味
- 强烈氨味：见于膀胱炎
- 丙酮气味：见于牛醋酮血症、生产瘫痪
### 透明度
- 正常：除马属动物浑浊不清外其他均为澄清透明
- 出现浑浊沉淀时提示存在白细胞、脓细胞、上皮细胞等
- 多见于肾脏、尿路的炎症
### 黏稠度
- 正常：马属动物尿液中含有粘液蛋白、核蛋白，故粘稠，而其他动物尿液稀薄
- 出现黏稠提示泌尿道的炎症，有形成分上升
### 尿色
- 正常：一般为黄色，受尿比重、饲料等影响
- 草食动物尿液呈碱性；肉食动物则呈酸性
##### 病理尿色
###### 血尿
- 尿液中含有红细胞，离心出现红细胞沉淀
- 因出血位置不同而呈现不同类型：

| 类型       | 排尿表现         | 常见病变部位            | 临床常见            |
| -------- | ------------ | ----------------- | --------------- |
| **初血尿**  | 排尿开始有血，后续转清  | **尿道、膀胱颈**        | 尿道损伤、尿道炎        |
| **终末血尿** | 排尿结束时出现血尿或血块 | **膀胱（尤其颈部/三角区）**  | 膀胱炎、结石、肿瘤       |
| **全程血尿** | 从开始到结束均为血色   | **肾脏、输尿管、广泛膀胱病变** | 肾炎、输尿管病变、膀胱弥漫病变 |
###### 血红蛋白尿
- 尿液中含有血红蛋白，颜色观与[[#血尿]]类似
- 镜下检查不含或少量含红细胞，提示溶血
- 隐血/潜血反应提示$\neq$血红蛋白尿(仅提示存在血红蛋白、红细胞或肌红蛋白)，需配合镜检
###### 胆色素尿
- 含有大量胆红素(以结合型为主)
- 提示肝、胆疾病
###### 脓尿
- 尿液中含有大量脓细胞
- 提示泌尿器官存在[[第四章 炎症#化脓性炎|化脓性炎症]]
###### 药物影响
- 深黄色$\to$内服四环素、土霉素、痢特灵等
- 红色$\to$百浪多息、酚红注射，内服安替比林、大黄等
## 泌尿器官检查
### 肾脏检查
##### 肾脏定位
- 不同动物不同

| 动物  | 相对位置             |
| --- | ---------------- |
| 牛   | 左肾会被瘤胃挤到右侧；右肾更固定 |
| 马   | 左右肾位置不对称，右肾更靠前   |
| 羊   | 左右肾都偏腰部，右肾比左肾更靠后 |
| 猪   | 左右肾几乎对称，位置接近     |
##### 检查方法
- 小动物：外部触诊
- 大动物：直肠检查
- 尿液检验、肾功能试验
##### 病理变化
- 检查的评估指标会围绕四个指标展开：$$大小+质地+压痛/敏感性+表面/波动感$$
因此我们会有：

| 触诊表现        | 最可能提示        | 病变类型   |
| ----------- | ------------ | ------ |
| 大 + 痛       | 急性炎症         | 炎症型    |
| 大 + 痛 + 波动感 | 积液、积脓        | 积液型    |
| 大 + 硬 + 结节  | 肿瘤、硬变、慢性结构破坏 | 硬变/肿瘤型 |
| 小           | 萎缩、发育不全、慢性病变 | 萎缩型    |
### 输尿管检查
经直肠检查输尿管
输尿管严重炎症$\to$输尿管紧张
严重的肾盂或输尿管结石$\to$触诊肾脏疼痛
### 膀胱检查
##### 位置确定
- 大动物：骨盆腔底部
- 小动物：不经直肠检查
##### 病理变化

| 类型         | 检查表现                | 常见原因/提示疾病                                    | 核心机制                     |
| ---------- | ------------------- | -------------------------------------------- | ------------------------ |
| **膀胱过度充满** | 膀胱明显膨大，触到充盈；有时受压可排尿 | 膀胱括约肌痉挛、膀胱麻痹、尿道阻塞（结石、肿瘤）、便秘压迫尿道、前列腺肥大或周围脓肿压迫 | **尿排不出去**，尿液在膀胱内潴留       |
| **膀胱空虚无尿** | 膀胱摸不到明显尿液充盈         | 肾功能高度障碍（肾衰竭、中毒）、膀胱破裂                         | **尿根本没生成**，或**生成后漏到膀胱外** |
| **膀胱压痛**   | 触压膀胱时动物疼痛、敏感        | 急性膀胱炎、膀胱结石、尿潴留                               | 炎症刺激或膀胱壁过度牵张、结石刺激        |
| **膀胱结石**   | 可触及结石；尿不太满时更易摸到     | 膀胱结石，且常伴尿潴留                                  | 结石在膀胱内形成，机械刺激膀胱壁并影响排尿    |
### 尿道检查
- 母畜：尿道炎
- 公畜：尿道结石、尿道损伤、尿道狭窄
