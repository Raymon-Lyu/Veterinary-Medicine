# 诊法
中医的诊断方式包括**望、闻、问、切**
辨证是对四诊获得的对病状和病情的认识，分析疾病的病因、部分、性质和证候
论治即是根据辨证的结果确立治疗的方法和原则
**察口色**和**切脉**是中兽医诊断的特色

## 望诊
### 望全身
1. 精神  内神在心，外神在目
2. 形体 形体与五脏相对应，形体瘦弱见于虚寒，形体强壮见于实热
3. 皮毛
4. 动态
### 望局部
1. 眼：瞳孔散大见于危重或中毒
2. 耳：
3. 鼻
4. 躯干与二阴等
### 察口色
- 口色的变化可以反映气血的盛衰以及脏腑的虚实
马属动物主要看唇舌、卧蚕和排齿，牛羊以舌底为最重要，猪主要看舌
- 主要观察色泽、口津、舌象(舌体和舌苔)
## 闻诊
- 通过听觉闻声音，嗅觉闻气味获取病情线索
