# 无菌术
## 概述
### 概念区别
* **无菌术**：是指应用消毒和灭菌的方法，==抑制或杀灭==可能到达手术区的病原微生物，从而预防手术创感染的综合措施 。
* **消毒**：选用适宜的化学药剂来杀灭病原微生物的方法 。
* **灭菌**：应用煮沸、蒸气等**物理方法**，杀灭手术器械、物品上的病原微生物的方法
### 技术手段
##### 物理性灭菌技术
主要依靠物理环境的改变来杀灭微生物，包含：
*   **热灭菌**：如高温煮沸灭菌、高压蒸气灭菌、干燥灭菌等
*   **紫外线灭菌**
##### 化学消毒技术
主要依靠化学药剂(冷消毒)，课件中列举了九大常用分类：

| 大类         | 具体分类       | Examples    |
| ---------- | ---------- | ----------- |
| **无机及元素类** | **酸与碱类**   | 醋酸、硼酸，氢氧化钠等 |
|            | **含氯类**    | 漂白粉、次氯酸钠    |
|            | **含碘类**    | 碘酊、碘伏、PVP碘等 |
|            | **过氧化物**   | 过氧乙酸、过氧化氢等  |
| **有机官能团类** | **醇类**     | 乙醇、异丙醇等     |
|            | **醛类**     | 甲醛、戊二醛等     |
|            | **酚类**     | 苯酚、煤酚和复合酚等  |
|            | **季胺盐类**   | 新洁尔灭        |
|            | **烷基化类气体** | 环氧乙烷等       |

## 器械与敷料的准备与消毒
### 分类准备
所有器械在消毒前应保持清洁，不得粘有污物或灰尘。根据物品材质的不同，适用的灭菌方法也有所区别：
- **金属器械**：可采用**高压蒸气灭菌法**、**化学药物浸泡消毒法**或**煮沸法**
- **玻璃、瓷和搪瓷类器皿**：采用**高压蒸气灭菌法**、**煮沸法**或**化学消毒药物浸泡法**。对于大件的器物（如大方盘、搪瓷盆等），可以考虑使用**酒精火焰烧灼灭菌法**
- **橡胶、尼龙和塑料类用品**：可选用**高压灭菌**、**煮沸灭菌**，或采用**化学消毒药液浸泡法**。此外，使用**环氧乙烷气体灭菌**既方便又简单。
- **敷料、手术创巾、手术衣和口罩等物品**：统一采用**高压蒸气灭菌**
### 常用消毒灭菌方法
##### 煮沸灭菌法
- **特点与用途**：简单方便，应用于多种物品的灭菌，但**要求速干的物品（如棉花、纱布、敷料等）除外**。
- **时间参数**：水沸腾3～5分钟后将器械放入煮锅内，待第二次水沸时开始计算时间。煮沸**15分钟**可以将一般的细菌杀灭，但不能杀灭细菌芽胞（杀灭芽胞需要煮沸**60分钟以上**）。
- **特殊配比**：在常水中加入碳酸氢钠使之成**2％的碱性溶液**，可以将其沸点提高至**102～105 ℃**。这不仅可以加强灭菌效果，还能防止金属器械生锈，但**注意该碱性溶液对橡胶制品有损害**。
##### 高压蒸气灭菌法
- **特点**：使用特制的灭菌器，压力高，温度增高。
- **核心参数**：通常蒸气压大约为**0.1～0.137MPa**左右，温度可达**121.6～126.6 ℃**。
- **时间与效果**：在此温压下维持**30分钟左右**，能彻底杀灭所有的细菌和芽胞。
- **用途**：广泛应用于各种物品的灭菌。
##### 化学药品消毒法
- **特点与用途**：不需要特殊设备，使用方便，常用于某些不宜用热力灭菌用品的消毒。但缺点是对细菌的芽胞往往难以杀灭。
- **临床常用药剂及参数**：
    - **新洁尔灭**：毒性较低，刺激性小，且消毒能力较强。常配制成**0.1％的溶液**，最常用来浸泡消毒器械、消毒手臂或其他可以浸湿的用品等。
    - **酒精**：常采用**70％的酒精**。可用于浸泡器械，**特别适于有刃的器械**，浸泡时间要求**不少于30分钟**。
    - **甲醛溶液**：常采用**10%浓度**。适用于金属器械、塑料薄膜、橡胶制品及各种导管的消毒，浸泡时间为**30分钟**。
    - **戊二醛**：常采用**2%水溶液**。用途与新洁尔灭相同，但灭菌效果更好，浸泡时间为**30分钟**。
    - **碘伏（聚乙烯酮碘）**：**7.5%溶液**用于消毒皮肤；**0.55%溶液**则以喷雾方式用于鼻腔、口腔、阴道粘膜的防腐。
##### 环氧乙烷气体消毒
- **理化特性**：环氧乙烷又名环氧乙烯，沸点为10.8℃。
- **优缺点对比**：优点是蒸汽压力大，穿透力强，具有很强的氧化性能；缺点是**易燃易爆**。
### 灭菌指示
- 在进行灭菌操作（如高压蒸气、环氧乙烷等）时，需配合使用**灭菌指示**（如指示胶带等）来验证灭菌过程是否达标成功。
## 手术人员准备
主要包括更衣、手与臂的清洁与消毒、穿着无菌手术衣以及戴手套四个环节：
### 更衣
- 在进行无菌准备前，必须先摘除所有的首饰。
- 换上专用的手术衣服，穿上套鞋，并规范戴好手术帽和口罩。
### 手、臂的清洁与消毒
这一环节是自我无菌化的核心，严格分为“洗刷”和“消毒”两个递进的步骤。
##### 手、臂的洗刷
- 使用肥皂进行规范的刷洗，在洗刷时应保持正确的姿势（手部保持朝上，避免脏水回流）。
- 用清水冲净后，必须使用无菌巾将手和臂完全拭干。
##### 手、臂的消毒
消毒完毕后有一个极其重要的姿势要求：**消毒过的手臂必须保持弯曲，将两手护放在胸前**。根据使用的药液不同，有以下三种标准的消毒流程：
###### 酒精浸泡法
- 使用百分之七十的酒精溶液浸泡五分钟。
- 随后使用百分之二的碘酊涂擦指甲边缘、指端等容易藏污纳垢的缝隙处。
- 最后再用百分之七十的酒精进行脱碘。
###### 新洁尔灭浸泡法
- 使用千分之一（即百分之零点一）的新洁尔灭溶液浸泡或拭洗五分钟。
- 完成后不需要擦拭，待其自然干燥并在皮肤上形成一层抗菌薄膜即可。
###### 聚乙烯酮碘涂刷法
- 使用聚乙烯酮碘药液涂刷五分钟。
- 用清水冲洗干净后，再次涂刷聚乙烯酮碘。
- 最后使用无菌巾彻底拭干。
### 穿着无菌手术衣
在完成上述手和臂的消毒工作（并在双手护胸的姿势下）后，按照无菌操作规范穿着无菌手术衣。
### 戴手套
穿好手术衣后即可戴上手套，临床操作中主要分为两种戴法：
##### 开放式戴法
手部先伸出手术衣的袖口，再进行戴手套操作。
##### 闭合式戴法
手部不伸出手术衣袖口，而是隐藏在袖管内，隔着无菌手术衣的袖口直接抓取并戴上手套，最大程度保证无菌状态。
## 病畜术部准备
在正式进入局部的术部准备之前，首先需要对患病动物进行一次全面的检查，并且在手术前应当对动物身体进行整体的清洁或洗刷。局部的术部准备主要分为以下三个核心步骤：
### 术部除毛
- **物理剃毛范围**：剃毛的范围必须足够大。通常要求超出切口周围二十至二十五厘米，如果是小动物，则可以控制在十至十五厘米的范围内。
- **化学脱毛剂**：除了常规的物理剃毛，也可以使用脱毛剂。课件中给出的具体配方为：$Na_2S$ 6.0至8.0g，加入蒸馏水100.0ml，混合制成溶液使用
### 术部消毒
- **常规用药与步骤**：临床上最常用的药物是5%的碘酊和70%的酒精。操作时，必须先用碘酊涂擦术部，然后**稍待片刻等其完全干透**，最后再使用70%的酒精将残留的碘酊擦去（即临床常说的脱碘过程）。
- **涂擦方向原则（极易考的实操细节）**：
    - 如果是**无菌手术**，涂擦方向应该是由手术区的**中心部向四周**扩散涂擦。
    - 如果是**已经感染的创口**，涂擦方向则正好相反，应该由**较清洁的区域（通常是外围）涂向感染的患处**。
 ### 术部隔离
- **操作方法**：完成术部皮肤消毒后，需要采用大块的、带有孔洞的手术巾覆盖在整个手术区域上，以此将消毒好的切口部位与周围未消毒的皮毛彻底隔离。
## 手术室环境管理与消毒
手术室需要有严格的管理制度来保证其洁净环境，除了日常的基础清洁外，还需要定期采取物理或化学的手段进行深层消毒。
### 手术室的常规管理与清洁制度
- **核心原则**：当一个手术室需要连续进行数个手术时，必须遵循**先做无菌手术，后做污染或感染手术**的原则。
- **日常清洁**：每次手术完毕后以及每天工作结束时，都应当彻底擦拭地面，将污液、废弃敷料和其他杂物全部清除。
- **定期大扫除**：除了日常打扫，每周还应彻底进行一次大扫除。
- **空间环境要求**：手术室需要保持良好的空气循环，空间应当密闭，并且保持正压或负压状态
### 空间与空气的定期消毒方法
手术室内必须定期进行空气消毒，临床上主要采用以下两大类技术：
##### 紫外光灯照射消毒（物理法）
- **杀菌效果**：紫外光的杀菌范围非常广，可以杀死一切微生物。紫外光灯的照射不仅可以有效地净化空气，明显减少空气中的细菌数量，同时也能杀灭手术室各物体表面附着的微生物。
- **使用要求**：必须在**非手术时间**开灯照射，才能达到明显的杀菌效果。
##### 化学药物熏蒸消毒（化学法）
这类方法的效果非常可靠且消毒彻底。操作时，必须先对手术室进行清洁扫除，然后关闭所有门窗做到较好的密封，最后再施以消毒药的蒸气进行熏蒸。临床最常用的有以下三种熏蒸配方：
- **甲醛加热熏蒸法**：按每立方米空间使用百分之四十的甲醛两毫升计算，加入等量的常水后进行加热蒸发。需要特别注意的是，消毒完成后必须让手术室通风排气，否则残留气体会有很强的刺激性。
- **甲醛配合高锰酸钾发烟法**：按照所用百分之四十甲醛溶液毫升数的一半来称取高锰酸钾。操作时，直接将高锰酸钾倒入甲醛溶液中，人员必须立刻退出手术室。仅仅数秒钟之后，两者反应便会产生大量烟雾状的甲醛蒸气，这种密闭消毒状态需要持续**四个小时**。
- **乳酸熏蒸法**：按每立方米空间使用百分之八十的乳酸十二毫升计算，加入等量的常水加热蒸发，蒸发产生的气体能有效杀灭空气中的细菌。整个蒸发过程持续六十分钟，加热完毕后需要将手术间封闭**四至六个小时**，杀菌效果同样非常可靠
