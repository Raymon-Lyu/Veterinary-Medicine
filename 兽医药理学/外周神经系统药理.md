## 神经系统的基本结构
- 突触，通过神经递质来调节细胞内外的离子浓度来传递信息
### 神经递质
- 胆碱能(Ach)神经：自主神经节前纤维，N受体，副交感神经节后纤维，⭐汗腺，骨骼肌血管舒张神经（只有交感），运动神经纤维
- 去甲肾上腺素能(NA/NE)神经：交感神经节后纤维主要是
- 多巴胺能神经：肾脏平滑肌、血管（交感节后纤维）
- NANC能神经
### 递质的代谢
- 乙酰胆碱(Ach)
		乙酰CoA+胆碱$\rightarrow$存于囊泡with ATP囊泡蛋白$\rightarrow$胞吐外排$\rightarrow$通过突触间隙胆碱酯酶(AchE)水解成乙酰CoA和胆碱
- 去甲肾上腺素
	主要依靠再摄取利用
### 递质的受体
- 胆碱能受体
	- M受体：毒蕈碱型，副交感神经节后纤维，有5种类型
	- N受体：神经节细胞和骨骼肌细胞
- 肾上腺素能受体
	- $\alpha$肾上腺素受体：平滑肌收缩（血管、瞳孔、膀胱）
	- $\beta$肾上腺素受体：
		- $\beta_1$，心肌细胞兴奋，心率和心收缩力增强
		- $\beta_2$，支气管和血管平滑肌扩张
		- $\beta_3$，脂肪细胞分解脂肪
## 传出神经药物的基本原理
[[兽医药理学/总论#1. 受体机制|参考总论受体部分]]
激动剂：结合受体后激活受体，产生与神经递质类似的效果，有内在活性
阻断剂：无内在活性，结合受体后不激活相关效应
作用方式：影响递质的合成、贮存、转运和分解消除

### 1. 氨甲酰胆碱 (Carbachol) 和 新斯的明 (Neostigmine) 药理作用的异同点

总述：

两者都是拟胆碱药 (Cholinergic agents)，都能产生乙酰胆碱 (ACh) 样效应，即兴奋胆碱能受体。但它们的作用机制和临床应用侧重有根本区别。

#### 相同点 (Similarities)

1. **最终效应：** 两者都能激动 M (毒蕈碱) 受体和 N (烟碱) 受体，表现出广泛的拟胆碱作用，例如：
    
    - **心血管：** 减慢心率、扩张血管、降低血压。
        
    - **消化道：** 增加胃肠道平滑肌的张力和蠕动。
        
    - **腺体：** 增加唾液、汗腺、支气管等腺体分泌。
        
    - **眼：** 瞳孔缩小、降低眼内压。
        
    - **骨骼肌：** 兴奋骨骼肌（N受体效应）。
        
2. **化学结构：** 它们都是**氨甲酸酯类**化合物。
    

#### 不同点 (Differences)

| **特征**        | **氨甲酰胆碱 (Carbachol)**                                                   | **新斯的明 (Neostigmine)**                                       |
| ------------- | ----------------------------------------------------------------------- | ------------------------------------------------------------ |
| **作用机制**      | **直接作用型**                                                               | **间接作用型（可逆性）**                                               |
| **详细机制**      | **直接激动** M 和 N 胆碱受体。                                                    | **抑制乙酰胆碱酯酶 (AChE)**，使内源性ACh在突触间隙堆积，从而间接激动M和N受体。              |
| **对AChE的稳定性** | **极难**被AChE水解，因此作用时间长且强。                                                | 作为AChE的底物，但其水解速度**极慢**（约30分钟-6小时），从而起到了抑制酶活性的作用。             |
| **主要激动受体**    | 同时激动 M 和 N 受体，对 **N 受体**的作用（尤其是对膀胱和胃肠道的N受体）相对更强。                        | 虽也激动 M 和 N 受体，但其对**骨骼肌 N$_{M}$ 受体**（神经肌肉接头）的作用尤为显著。          |
| **临床应用侧重**    | 1. **青光眼**：滴眼以缩瞳、降眼压。 2. **胃肠道/膀胱无力**：用于术后肠麻痹或尿潴留（因其对胃肠道和膀胱的N受体有强兴奋作用）。 | 1. **重症肌无力**（主要用途）。 2. **非去极化肌松药的拮抗剂**（麻醉后苏醒）。 3. 术后肠麻痹和尿潴留。 |
| **中枢神经系统**    | 是季铵盐，**不易**通过血脑屏障。                                                      | 是季铵盐，**不易**通过血脑屏障。                                           |

总结：

最大的区别在于作用机制。

- **氨甲酰胆碱**像一把“万能钥匙”，直接去开 M 和 N 两种锁。
    
- **新斯的明**则是“锁匠”，它通过抑制AChE，阻止了ACh被降解，让ACh（“原配钥匙”）的数量大大增加，从而间接实现了开锁效应。

### 2. 甲溴东莨菪碱 + 硫酸新霉素 联合用药的药理基础

这种联合用药在兽医学（特别是治疗动物的胃肠道疾病，如腹泻和肠炎）中较为常见。其药理基础是一种典型的**“对因”与“对症”相结合**的治疗策略。

#### 1. 硫酸新霉素 (Neomycin Sulfate)：对因治疗（抗感染）

- **药物类别：** 氨基糖苷类抗生素。
    
- **药理作用：** 通过抑制细菌的蛋白质合成来杀灭敏感菌。
    
- **药代动力学关键点：** **口服后几乎不被胃肠道吸收**。
    
- 在联合用药中的角色：
    
    由于口服不吸收，硫酸新霉素能以高浓度停留在胃肠道内，直接杀灭或抑制引起胃肠道感染（如细菌性肠炎、腹泻）的病原菌。它充当了病因治疗的角色。
    

#### 2. 甲溴东莨菪碱 (Methscopolamine Bromide)：对症治疗（解痉、止泻）

- **药物类别：** 抗胆碱药（M受体阻断剂）。
    
- **药理作用：** 阻断 M 胆碱受体，与乙酰胆碱竞争，从而产生“抗胆碱”效应。
    
- **在联合用药中的角色：**
    
    1. **抑制腺体分泌：** 减少胃肠道内异常增多的分泌物（如肠液、粘液），使粪便成形，缓解水样泻。
        
    2. **抑制平滑肌：** 缓解胃肠道平滑肌的痉挛和过度蠕动，从而**止痛**（缓解腹部绞痛）和**止泻**（减慢肠内容物推进速度，增加水分重吸收）。
        
    3. **结构特点：** 它是东莨菪碱的季铵盐衍生物，**不易通过血脑屏障**，因此中枢副作用（如镇静、幻觉）比阿托品或东莨菪碱轻微。
        

#### 联合用药的协同基础

**硫酸新霉素**负责消除**病因**（细菌感染），而**甲溴东莨菪碱**负责缓解由感染引起的**症状**（腹泻、腹痛、肠痉挛）。

两者联用，可以：

1. **快速控制症状：** 甲溴东莨菪碱迅速缓解动物的腹痛和腹泻，改善其状态。
    
2. **根本消除病原：** 硫酸新霉素在肠道内清除致病菌。
    
3. **协同增效：** 甲溴东莨菪碱减慢了肠道蠕动，这在客观上**延长了硫酸新霉素在肠道内的停留时间**，使其能更充分地与细菌接触，从而可能增强其抗菌效果。
    

这种组合拳式的用药方式，可以同时解决感染和症状两大问题，达到标本兼治的效果。
