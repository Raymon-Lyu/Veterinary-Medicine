### 分泌机制和调节机制
肾上腺分为皮质和实质，皮质又可以分为球状带、束状带、网状带，各自对应的激素为：

| 分泌部位 | 分泌激素                    |
| ---- | ----------------------- |
| 球状带  | 盐皮质激素(醛固酮)              |
| 束状带  | 糖皮质激素                   |
| 网状带  | 性激素                     |
| 髓质   | [[外周神经系统药理#肾上腺素\|肾上腺素]] |
上述激素的调节存在内源性的**下丘脑-垂体-分泌腺**机制
