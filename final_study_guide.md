# 第一板块：总论概念速背（填空 + 名词解释 + 判断）

## 1. 核心定义

| 概念 | 背诵要点 |
|------|---------|
| **传染** | 病原微生物侵入机体，在一定部位定居繁殖，并产生病理反应的过程 |
| **传染病** | 由**病原微生物**引起，具有一定**潜伏期和症状**，并具有**传染性**的疾病 |
| **传染病四大特征** | ①由病原微生物引起 ②具有传染性和流行性 ③被感染动物可产生特异性免疫 ④出现临床症状 |

## 2. 感染类型辨析（判断题重灾区！）

| 类型 | 定义 | 辨析陷阱 |
|------|------|---------|
| **显性感染** | 致病力 > 抵抗力，有病理变化+**有**临床症状 | — |
| **隐性感染** | 致病力 ≈ 抵抗力，有病理变化+**无**临床症状 | 隐性感染≠不感染，仍可排毒！ |
| **温和型** | 症状完整但程度较轻 | 属于**显性感染**的一种！ |
| **非典型** | 典型症状缺失，易造成误诊 | 同属**显性感染** |
| **顿挫型** | 出现前驱症状后迅速停滞恢复 | — |

## 3. 传染病发展四个阶段

```
潜伏期 → 前驱期 → 明显期(发病期) → 转归期
```
- **潜伏期**：病原存在繁殖但不表现症状（可能排毒！传染期≠发病期）
- **前驱期**：出现**非特异性**症状
- **明显期**：出现该病**固有/特征性**症状
- **转归期**：康复或死亡

> 防疫决策口诀：**确诊观察一个传染期，疑似观察一个最长潜伏期**

## 4. 流行过程三环节

| 环节 | 定义 | 关键判断 |
|------|------|---------|
| **传染源** | 体内有病原定殖 + **向外排出病原** | 感染死亡的尸体**不是传染源**，是传播途径！ |
| **传播途径** | 病原从传染源排出后侵入易感动物的途径 | 分直接接触/间接接触；间接又分**媒介者**（活物）和**媒介体**（死物） |
| **易感动物** | 由物种、品种、年龄（内在）+ 环境管理（外在）+ 免疫状态（后天）决定 | — |

**传播方式辨析**：
- **水平传播**：同一代个体间传播
- **垂直传播**：经**卵巢、子宫、初乳**在两代间传播
- **粪口传播**：病原随粪便排出→污染饲料/饮水→经消化道感染
- **甲型流感、乙脑等**→ 经**蚊虫叮咬生物性媒介者**传给人

## 5. 全身感染类型辨析

| 类型 | 核心区别 |
|------|---------|
| **菌/病毒血症** | 病原**短暂进入血液**，不在血中繁殖，只是"搭地铁"去靶器官 |
| **毒血症** | 病原留在局部，但其**外毒素**进入血液毒害全身 |
| **败血症** | 病原大量侵入血液并在血中**大量繁殖**，产生毒素 |
| **脓毒血症** | 败血症进阶形态，**化脓菌**经血液形成多发性转移性脓肿 |

## 6. "因于病毒，死于细菌"

= 传染病学中的**继发感染**现象。原发病毒（如PRRSV/CSFV）破坏免疫系统→继发细菌（副猪嗜血杆菌/链球菌）大量繁殖→动物实际死于细菌感染。

## 7. 防疫措施

- **方针**：预防为主，养防结合
- **疫病控制两个阶段**：①日常**预防**（自繁自养、检疫、消毒、接种） ②发病时**扑灭**（诊断上报、隔离、封锁、终末消毒）
- **控制三个关键点**：消灭传染源 + 切断传播途径 + 提高易感动物抵抗力

## 常用检测方法（填空）

| 疾病 | 常用诊断方法 |
|------|------------|
| **鸡白痢** | **平板凝集试验** |
| **牛布氏杆菌病** | **虎红平板凝集试验 / 试管凝集试验(SAT)** |
| **炭疽** | **Ascoli环状沉淀试验**（不能解剖！涂片镜检竹节状串链） |
| **新城疫** | **HI（血凝抑制）试验** + ICPI致病指数测定 |
| **禽流感** | **RT-PCR** 分型 HA/NA |

## 梭菌相关疾病

- 梭菌引起羊——**快疫/肠毒血症**
- 梭菌引起猪——**仔猪红痢（C型产气荚膜梭菌）**、**仔猪梭菌性肠炎**
- 梭菌最重要的特点——**产生强力外毒素** + 芽孢抵抗力极强

## 沙门氏菌猪 vs 禽的差异

- 猪：猪霍乱沙门氏菌引起**仔猪副伤寒**（大肠糠麸样坏死性肠炎）
- 禽：鸡白痢/禽伤寒沙门氏菌（主要致病因子为**内毒素 LPS**，不产肠毒素）
- 猪沙门氏菌与禽不同的致病因子：猪还存在**Vi抗原（荚膜）**增强侵袭力

## 非洲猪瘟 (ASF) 核心记忆点

- **唯一**已知的虫媒DNA病毒（非洲猪瘟病毒科，大DNA病毒）
- 传播媒介：**钝缘蜱**（生物性传播）
- **无疫苗、无有效治疗药物**
- 确诊后必须**全群扑杀 + 无害化处理**
- 诊断依据：红细胞吸附试验 (HAD)、PCR

## 炭疽三要素（每年必考）

1. **尸体特征**：尸僵不全 + 天然孔出血 + 血液凝固不良呈暗红色煤焦油样
2. **诊断方法**：Ascoli环状沉淀试验；涂片见竹节状串链菌体
3. **绝对禁忌**：不能随意**解剖**（芽孢暴露空气后可存活数十年污染土壤）

# 第三板块：三大简答满分答案（两卷100%重叠！）

## 📝 简答1：三种禽免疫抑制病

### ① 马立克氏病 (MD)
- **病原**：MDV（疱疹病毒科）
- **易感动物**：雏鸡最易感
- **传播途径**：经**羽毛囊皮屑**随空气呼吸道水平传播。**不经卵垂直传播**
- **临床与病变**：神经型"大劈叉"姿势 + 眼型虹膜"灰眼" + 内脏灰白肿瘤结节 + 坐骨神经肿大横纹消失 + 法氏囊**萎缩**

### ② 传染性法氏囊病 (IBD)
- **病原**：IBDV（双RNA病毒科）
- **易感动物**：**3-6周龄**鸡
- **传播途径**：消化道、呼吸道水平接触
- **临床与病变**：突然发病、尖峰死亡；排白色蛋清样稀粪；法氏囊"**紫葡萄**"→后期萎缩；**胸肌腿肌条状出血**

### ③ 鸡传染性贫血 (CIA)
- **病原**：CIAV（环状病毒科）
- **易感动物**：**1-3周龄**雏鸡
- **传播途径**：高度**经卵垂直传播** + 水平接触
- **临床与病变**：极度苍白、血稀如水、"**蓝翅病**"；骨髓萎缩变**淡黄色脂肪样**；胸腺极度萎缩

---

## 📝 简答2：三种引起禽产蛋下降的病

### ① 新城疫 (ND) — NDV（副黏病毒科）
- 产蛋急剧下降，大量**软壳蛋、褪色蛋、畸形蛋**
- 呼吸困难、嗉囊酸臭液、排黄绿色稀粪、扭颈观星
- 剖检：腺胃乳头出血；盲肠扁桃体及肠道**枣核状坏死溃疡**

### ② 禽流感 (AI) — AIV（正黏病毒科）
- 低致病性：产蛋下降，产白壳蛋/沙皮蛋，轻微呼吸道
- 高致病性：肿头流泪、鸡冠肉垂发绀坏死、脚鳞出血、猝死
- 剖检：广泛内脏出血；**卵泡充血破裂→卵黄性腹膜炎**

### ③ 鸡传染性支气管炎 (IB) — 传染性支气管炎病毒 (IBV，冠状病毒科)
- **临床与产蛋下降症状**：
  - 产蛋率显著下降，大量产出**畸形蛋、软壳蛋和粗壳蛋**，最特征病变为**蛋清稀薄如水**（蛋清与蛋黄分离）。
  - 伴随气管啰音、喘息等轻微呼吸道症状，或拉白色稀粪。
  - **雏鸡阶段，特别是1-14日龄内**感染会导致输卵管发生永久性不可逆损伤，成年后表现为不产蛋的**企鹅样假母鸡**。
- **特征剖检**：气管/支气管交界处常见黄白色干酪样栓子；肾型感染表现为肾脏肿大苍白、尿酸盐沉积（**花斑肾**）。

## 📝 简答3：断奶仔猪(45-60日龄)呼吸道 + 败血症鉴别

### 🧬 病毒性（任选2种）

**① PRRSV / 蓝耳病**
- 高热(40.5-41.5℃)，腹式呼吸，犬坐姿势
- 皮肤发绀（**耳尖、腹下蓝紫色**）
- 肺脏褐色肝变（间质性肺炎，间质增宽）

**② CSFV / 猪瘟**
- **稽留热**，结膜齿龈发绀，便秘腹泻交替
- 剖检：**花斑肾**(出血)、**脾脏边缘梗死**、**盲肠纽扣样溃疡**

### 🧫 细菌性（任选2种）

**① 副猪嗜血杆菌 (HPS / 格拉泽氏病)**
- 关节肿大跛行 + 发热咳嗽
- 剖检：典型"**包心包肝**"——心包腔、腹膜腔、胸膜腔充满**纤维素性渗出物**

**② 传染性胸膜肺炎 (APP)**
- 突发高热，呼吸极度困难，**口鼻流泡沫样带血分泌物**，犬坐姿势
- 剖检：肺脏**纤维素性出血性胸膜肺炎**（肺叶粘连胸壁），气管内有血色泡沫

# 第五板块：分析题思路框架

## 猪瘟兔化弱毒实验逻辑

核心原理：
- **强毒**对兔无症状、体温不升高，但能产生免疫→再打弱毒不升温
- **弱毒(C株/兔化弱毒)**使兔体温**升高**（定型热反应）

| 实验结果 | 判断结论 |
|---------|---------|
| 实验组不升温 → 对照组升温 | 病料含**猪瘟强毒**（产生免疫，抵挡了弱毒） |
| 都不升温 | 病料不含猪瘟病毒（**也不含其他对兔有作用的病原**） |
| 都升温 | 病料不含猪瘟，但含有**对兔有作用的其他病原**（两组都发热） |
| 实验组之前升、之后也升 | 病料含**猪瘟弱毒**（第一次弱毒使其升温，第二次打的也是弱毒，再次升温） |
| 对照组不升 | 对照组只接种弱毒却不升温→说明弱毒被中和→**不该出现此情况**（或弱毒失效） |

## PEDV场景分析思路

1. **胃肠疾病送检病原**：PEDV、TGEV、猪轮状病毒（还可检ETEC、沙门氏菌、猪瘟等）
2. **基因型≠血清型**：同一血清型内可有不同基因型，基因型差异导致抗原位点改变→疫苗株与流行株抗原不匹配→**交叉保护失败**
3. **免疫失败原因**：①疫苗毒株与流行毒株基因型/抗原不匹配 ②反饲操作不规范（病毒灭活/剂量不足） ③母源抗体未有效传递给仔猪
4. **改进措施**：更换与当地流行株基因型匹配的疫苗；加强母猪产前免疫程序；改善生物安全（消毒、全进全出）
