# 猪病汇总

| 序号     | 疾病名称                | 概述 / 病原                           | 流行病学                                   | 症状                                                                        | 病理变化                                          | 诊断                                                 | 防治                                   |     |
| :----- | :------------------ | :-------------------------------- | :------------------------------------- | :------------------------------------------------------------------------ | :-------------------------------------------- | :------------------------------------------------- | :----------------------------------- | --- |
| **1**  | **猪瘟 (CSFV)**       | 由黄病毒科瘟病毒属的单链RNA病毒引起，民间称“烂肠瘟”。     | 可经水平与垂直途径传播，60日龄前后最易发生。                | 1. 潜伏期一般9天<br>2. 呈稽留热<br>3. 结膜及齿龈发绀<br>4. 便秘与腹泻交替<br>5. 繁殖障碍型,见新生仔猪“抖抖病”。 | 盲肠、扁桃体见典型的“纽扣样溃疡”，脾脏梗死，肾脏/膀胱出血，肋弓骨质发育障碍。      | 临床诊断结合白细胞计数，荧光或酶标抗体染色、兔体交互免疫或ELISA。                | 采用疫苗免疫接种(0-30-60日龄)，一旦发病需扑杀淘汰。       |     |
| **2**  | **猪繁殖与呼吸综合征 (蓝耳病)** | 由动脉炎病毒引起，主要感染单核/巨噬细胞，造成免疫抑制,。     | 病毒可经空气传播，也能穿透胎盘垂直感染胎儿。                 | 表现为母猪“流产风暴”和并发呼吸道继发感染，曾引发大规模“高热病”,。                                       | (讲义侧重其临床引发的继发症表现，未详述特异性肉眼病变)。                 | 临床诊断结合血清学检测，利用PCR区分强毒与弱毒。                          | 灭活疫苗免疫效果不佳，弱毒疫苗的使用需谨慎。               |     |
| **3**  | **伪狂犬病**            | 由猪疱疹病毒I型引起，易造成猪只的持续感染。            | 猪及猫、犬等多种动物易感，可通过垂直与水平途径传播。             | 成年猪多隐性，母猪表现繁殖障碍，新生仔猪出现神经症状及高死亡率，犬猫表现为奇痒。                                  | (讲义侧重其严重临床表现，未特别强调肉眼病理变化)。                    | 临床诊断结合血清学检测（区分总抗体与基因缺失抗体）及PCR。                     | 依赖基因缺失疫苗免疫及开展该病的净化消灭计划。              |     |
| **4**  | **猪传染性胃肠炎 (TGE)**   | 由冠状病毒(TGEV)引起，病毒对热和阳光敏感。          | 具有明显季节性（多发于冬季或寒潮诱发），通过污染饮食传播，吃奶乳猪受害最重。 | 潜伏期极短(15-18h)，乳猪表现为水样腹泻、呕吐并迅速脱水死亡；大猪常为3~5天的一过性发病。                         | 剖检可见小肠壁变菲薄、明显充血，肠内充满水样内容物。                    | 临床虽易发现，但与流行性腹泻极难区分，需依赖实验室诊断鉴别。                     | 使用疫苗（活苗/灭活苗）免疫，加强冬季饲养管理的保温防寒。        |     |
| **5**  | **猪流行性腹泻 (PED)**    | 由冠状病毒(PEDV)引起，与TGEV无明显抗原交叉。       | 具有明显季节性（多发于冬季或寒潮诱发），吃奶乳猪受害最大。          | 症状与TGE极其相似，表现为剧烈的水样腹泻、呕吐、脱水。                                              | 病变与TGE类似，主要表现为小肠壁菲薄充血。                        | 临床难以与传染性胃肠炎区分，需实验室病原学鉴别诊断。                         | 同TGE，主要依赖疫苗免疫和改善环境温度。                |     |
| **6**  | **猪丹毒**             | 由猪丹毒杆菌（革兰氏阳性小杆菌）引起的传染病,。          | 呈散发，猪、人及其他部分禽类、鱼类均易感。                  | 急性型突发高热、粪便干结、败血死亡；亚急性型皮肤出现特征性“打火印”疹块；慢性型表现关节炎。                            | 急性期呈全身败血症，慢性期可见关节炎、典型的疣状心内膜炎和皮肤坏死。            | 疹块型易于临床诊断，可进行青霉素治疗试验或涂片染色镜检。                       | 青霉素治疗具有特效，日常可通过常规疫苗进行免疫。             |     |
| **7**  | **猪肺疫 (锁喉风)**       | 属于巴氏杆菌病，由多杀性巴氏杆菌引起，镜检呈“两极浓染”,。    | 极易在急应激后发生，也常继发于猪气喘病等呼吸道疾病。             | 突发急性高热，颈下咽喉部发生严重炎性肿胀、发烫，呼吸困难导致口鼻流出白沫。                                     | 病理学表现主要为严重的肺炎、组织出血及水肿。                        | 需与猪瘟、猪丹毒、猪链球菌病进行严格的临床鉴别诊断。                         | 使用抗菌药治疗，常接种猪瘟、猪丹毒、猪肺疫三联疫苗预防,。        |     |
| **8**  | **猪链球菌病**           | 由致病性链球菌（如2型猪链球菌、C群兽疫亚种）引起。        | 易感动物广，特别是2型具有极高的公共卫生风险（人感染易致死）。        | 表现型极多，涵盖淋巴结脓肿、急性败血、脑膜脑炎、多发性关节炎及胸膜肺炎等,。                                    | 病变可见于肺脏、肾脏、膀胱、脾脏、脑部及各种浆膜和粘膜面。                 | 临床易与其他急热性传染病混淆，确诊需依靠细菌分离鉴定。                        | 早期并足疗程使用有效抗菌药物，预防依靠弱毒或灭活疫苗免疫。        |     |
| **9**  | **猪大肠杆菌病**          | 由带粘附素和肠毒素（LT/ST）的致病性大肠杆菌(ETEC)引起。 | 分为7日龄内的黄痢、7日龄至断奶前的白痢、以及断奶应激后诱发的水肿病,。   | 黄痢排黄色腥臭粪，白痢排水泥浆样白粪；水肿病常出现神经症状和叫声异常,。                                      | 肠道充满特征性稀粪并伴随充出血；水肿病则表现为眼睑、颈下、胃大弯及肠系膜水肿,。      | 结合发病日龄及临床症状诊断，实验室需检测LT毒素或粘附素。                      | 母猪产前免疫，确保仔猪吃足初乳，改善环境卫生，水肿病需提早诱食控料。   |     |
| **10** | **猪沙门氏杆菌病 (仔猪副伤寒)** | 由猪霍乱沙门氏杆菌引起的疾病，该菌也是猪瘟的常见继发菌。      | 发病集中在断奶前后，且倾向于在阴雨连绵的寒冷季节群发。            | 急性型表现为高热败血，耳朵和腹下皮肤发绀，伴随腹泻。                                                | 典型病变为脾脏肿大，肠系膜淋巴结肿大呈索状，大肠黏膜出现糠麸样坏死性肠炎。         | 依靠临床诊断并结合实验室的细菌分离鉴定。                               | 使用相应的疫苗预防，或进行药物预防和治疗。                |     |
| **11** | **猪气喘病**            | 由猪肺炎支原体引起的慢性呼吸道病,。                | 多呈隐性带毒，经呼吸道传播，受气候影响极大且极易继发加重。          | 病程发展缓慢，表现为慢性的长期咳嗽和气喘，严重影响猪只生长。                                            | 肺脏出现特征性的融合性支气管肺炎（呈“肉变”），肺门淋巴结肿大。              | 以全群为单位观察，结合X光检查，并可设立不接种的站岗猪进行传染性观察。                | 在饲料中添加支原净、泰乐菌素，或使用氟喹诺酮类治疗；日常需建立健康群。  |     |
| **12** | **猪传染性萎缩性鼻炎**       | 由支气管败血波特氏杆菌和产毒素巴氏杆菌混合引起。          | 疾病发病与传播缓慢，大约克猪、长白猪相对更加易感。              | 早期表现鼻腔出血、打喷嚏发痒；中后期发展为面部和鼻子变形（弯曲），眼角出现“泪斑”。                                | 切开第一臼齿横断面，可见特征性的鼻甲骨萎缩。                        | 临床观察面部及病理切面，实验室通过PCR检测产毒素巴氏杆菌。                     | 通过药物添加或疫苗（注意副作用）免疫，并淘汰病猪建立健康无病群。     |     |
| **13** | **猪细小病毒病**          | 由细小病毒(PPV)引起，病毒具有红细胞血凝特性。         | 核心侵害头胎怀孕母猪及其胎儿，主要由公猪带毒传播。              | 导致母源性繁殖障碍，但该病毒对公猪的受精率和性欲没有明显影响。                                           | 病变局限在生殖系统，产出死胎、木乃伊胎等。                         | 结合“仅影响头胎母猪、公猪无碍”的临床特点诊断，实验室做HA/HI及胎儿荧光染色。          | 进行疫苗免疫接种，或让头胎母猪在配种前合群自然感染以产生免疫力。     |     |
| **14** | **口蹄疫 (共患)**        | 由口蹄疫病毒（小RNA病毒）引起的烈性传染病，属OIE重点疫病。  | 偶蹄兽（如猪、牛）极易感，经空气等间接接触传播，气温低的季节多发。      | 潜伏期短，口腔、蹄部和乳房部位出现水泡和溃疡。                                                   | 除水泡溃疡外，死亡的幼龄动物（乳猪）剖检可见典型的“虎斑心”（心肌坏死）。         | 临床易诊，需结合实验室（送检水泡皮/液、PCR、ELISA）以及动物接种试验（如小鼠）进行鉴别定型。 | 采取“免疫+扑杀”与严格检疫的综合政策，免疫以灭活疫苗为主。       |     |
| **15** | **日本乙型脑炎 (共患)**     | 由黄病毒科的乙脑病毒（虫媒病毒）引起的共患病。           | 猪、马、人等多动物易感，高度依赖蚊子进行增殖和传播，仅在夏季发生。      | 猪感染常突发稽留高热，母猪发生流产/死胎/木乃伊胎，公猪呈单侧睾丸肿胀变硬。                                    | 脑、脊髓充出血和积液，子宫内膜炎，公猪睾丸病变，流产胎儿见脑水肿及腹水增多。        | 结合明显的夏季高发及繁殖障碍特点，实验室进行HA、血清中和试验、荧光抗体或PCR。          | 核心在于防蚊灭蚊，并根据季节对种群进行免疫接种。             |     |
| **16** | **破伤风 (共患)**        | 由严格厌氧的破伤风梭菌在体内繁殖产毒（痉挛毒素）引起。       | 猪、马、牛等均易感，主要通过创伤（如阉割、断脐）在厌氧环境下引发散发感染。  | 症状从头部开始，对外界刺激敏感兴奋，发生严重的肌肉痉挛和骨骼肌强直收缩。                                      | 病变主要是创伤局部的厌氧毒血症发作（缺乏典型的全身肉眼病理改变）。             | 根据受伤病史及骨骼肌强直等典型临床症状进行确诊。                           | 重视外伤处理，使用类毒素预防或抗毒素被动免疫封闭，发病后镇静解痉并护理。 |     |
| **17** | **炭疽 (共患)**         | 由炭疽杆菌引起的古老且极度危险的人畜共患败血症。          | 食草动物、杂食动物（含猪）以及人类皆易感，污染土壤为重要传染源。       | 猪等动物发病常表现为急性的出血性败血症。                                                      | **死亡尸体严禁剖解**；特征为皮下黄色胶冻样浸润、血液凝固不全、脾脏极度肿大且结构模糊。 | 依据死亡尸体特征，严格采血涂片染色镜检，或通过青霉素串株试验确诊。                  | 发病时青霉素治疗特效，高发区接种炭疽芽孢苗，尸体必须严格化制或焚烧销毁。 |     |
| **18** | **布氏杆菌病 (共患)**      | 由布氏杆菌（有牛、羊、猪型等多种分型）引起的共患病。        | 各类家畜广泛易感，极易感染人类，呈缓慢发展过程。               | 主要侵害生殖器官，怀孕母猪发生传染性流产，公猪患睾丸炎导致不孕，偶见关节炎。                                    | 肉眼病变主要局限在生殖系统，如胎盘的充出血与水肿、睾丸及乳腺的炎症等。           | 临床结合免疫学方法，如血清试管凝集试验（SAT）进行筛查诊断。                    | 每年加强定期的检疫和血液监测，发现阳性坚决淘汰，培育健康群并严格消毒。  |     |
