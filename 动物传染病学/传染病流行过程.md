# 传染病流行过程
在[[动物传染病学/总论|总论]]中，我们更多关注
- 流行三要素：**传染源**、**传播途径**、**易感动物**
### 传染过程
##### 传染源
- 定义：机体内存在病原定殖，且==向外排出(释放病原)==，不指向病原
传染源要满足病原定殖+向外排出病原的**2**个条件
如闭锁性结核不具有传染性，**感染死亡的尸体不是传染源，是传播途径**
因此，可以将传染源大致分为两类：
1. 发病动物：位于传染期
2. 带菌(毒)动物：潜伏期、转归期、健康期+向外排毒
> 对于carrier，只有完成**微生物学康复**才意味着脱离传染期，而非临床康复和病理学康复
##### 传播途径
- 定义：病原从传染源排出后，侵入易感动物的途径
具体途径可分为：
- 直接接触传染：如狂犬病
- 间接接触传染：通过媒介传播$\to$媒介者/媒介体
在间接接触传染途径中又可以将媒介分为媒介者和媒介体：

| 媒介者    | 媒介体      |
| ------ | -------- |
| 活物     | 死物       |
| 病原可能爆发 | 病原数量被动衰减 |
| 主动移动   | 被动位移     |
| 生态灭杀   | 理化消杀     |

> - 间接传播：病原在同一代个体间进行传播
> - 垂直传播：病原通过==卵巢、子宫、初乳==在两代间进行传播
##### 易感动物
决定动物是否易感主要由以下三个方面决定：
1. 内在因素：物种、品种、年龄
2. 外在因素：卫生环境与饲养管理
3. 后天特异性免疫：[[疫苗]]
