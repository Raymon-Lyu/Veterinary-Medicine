# 传染病流行过程
在[[动物传染病学/总论|总论]]中，我们更多关注
- 流行三要素：**传染源**、**传播途径**、**易感动物**
### 传染过程
##### 传染源
- 定义：机体内存在病原定殖，且==向外排出(释放病原)==，不指向病原
传染源要满足病原定殖+向外排出病原的**2**个条件
如闭锁性结核不具有传染性，**感染死亡的尸体不是传染源，是传播途径**
因此，可以将传染源大致分为两类：
1. 发病动物：位于传染期
2. 带菌(毒)动物：潜伏期、转归期、健康期+向外排毒
> 对于carrier，只有完成**微生物学康复**才意味着脱离传染期，而非临床康复和病理学康复
##### 传播途径
- 定义：病原从传染源排出后，侵入易感动物的途径
具体途径可分为：
- 直接接触传染：如狂犬病
- 间接接触传染：通过媒介传播$\to$媒介者/媒介体
在间接接触传染途径中又可以将媒介分为媒介者和媒介体：

| 媒介者    | 媒介体      |
| ------ | -------- |
| 活物     | 死物       |
| 病原可能爆发 | 病原数量被动衰减 |
| 主动移动   | 被动位移     |
| 生态灭杀   | 理化消杀     |

> - 间接传播：病原在同一代个体间进行传播
> - 垂直传播：病原通过==卵巢、子宫、初乳==在两代间进行传播
##### 易感动物
决定动物是否易感主要由以下三个方面决定：
1. 内在因素：物种、品种、年龄
2. 外在因素：卫生环境与饲养管理
3. 后天特异性免疫：[[疫苗]]
### 不知道写什么
##### 疫源地
- 定义：[[#传染源]]存在+病原排出污染环境
- 不依赖人类活动而存在的疫源地成为**自然疫源地**
- 自然疫源特征
##### 疫点&疫区
- 定义：传染病正在发生的区域`余老师version`
- 与[[#疫源地]]一样存在时间延续性(*传染源没了，但是环境存在病原*)，也存在空间扩展性(如3km直径疫区+5km扩展威胁区)
- 疫区作为行政命令单位

### 流行过程的特征
##### 流行强度
- 指流行的过程中，发病率的高低和流行范围的大小
流行强度与三个变量相关：**时间**、**空间**、**发病数量**，因此有：
1. 散发性
2. 地方流行性
3. 流行性
4. 大流行性
##### 季节性和周期性
季节性是以年为循环，是自然环境中温度和湿度的变化对[[#传染过程|“三大环节”]]产生了影响：
- 传染源：“冷”是最好的保鲜剂，病原可以保持活性几个月
- 传播途径：夏天蚊虫繁殖增加，以此为媒介者的传播途径被加强，如乙脑
- 易感动物：冬天营造的密闭环境提高了呼吸道感染的风险
周期性与**群体免疫阈值**相关：
- 病原变异导致毒力增强，如[[RNA病毒#甲型流感病毒|流感病毒]]
- 免疫力的衰退([[适应性免疫应答#回忆应答|免疫记忆产生]])与种群更替(无“抗体”个体产生)
### 流行过程影响因素
##### 自然因素
影响[[#传染过程]]各环节，如地理隔离阻断感染源，气候温度限制蚊虫
##### 社会因素
社会因素会改变某些自然因素的限制
如将泔水喂猪造成猪瘟的爆发，以及兽医执法力度对易感行为的限制
社会因素强调了影响流程过程的核心是人

### 流行病学调查与分析
为了了解传染病的发生原因、传播条件和传播动向，需要对[[#传染过程|传染三大过程]]、疫区范围、发病率、死亡率蹬进行调查分析
调查的方法有：
1. 询问调查
2. 现场观察
3. 实验室检查
4. 统计学方法：发病率&患病率、死亡率&病死率、感染率&带菌率
