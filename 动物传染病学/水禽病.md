# 水禽核心传染病鉴别诊断纲要
**【使用说明】** 本纲要专为应对客观题（选择/判断）设计，剥离长篇机制，绝对聚焦于**“唯一性特征（题眼）”**。考场上看到加粗关键词，直接锁定对应疾病。

---

## 一、 病毒性疾病：雏禽杀手与产蛋毁灭者

### 1. 鸭病毒性肝炎 (DVH)
*   **致病原：** 鸭甲型肝炎病毒 (DHAV，微小核糖核酸病毒科)。
*   **易感日龄：** 严格受限于 **1~3 周龄** 雏鸭，大鸭隐性感染不发病。
*   **临床特征题眼：** 发病急，死前严重抽搐，标志性死亡姿势为 **“角弓反张” (背脖子、双腿后伸)**。
*   **解剖特征题眼：** 肝脏肿大，表面布满 **鲜红色出血点/斑/刷状出血**。
*   **特效急救：** 紧急皮下注射抗鸭肝炎**高免卵黄抗体**。

### 2. 鸭呼肠孤病毒感染 (DRV)
*   **致病原：** 鸭呼肠孤病毒（正呼肠孤病毒科，无包膜，可垂直传播）。
*   **易感日龄：** 雏鸭多发（番鸭、樱桃谷鸭等），常伴发**跗关节肿大跛行**。
*   **解剖特征题眼 (坏死双绝)：**
    *   **脾坏死 (花脾/大白脾)：** 脾脏极度肿大，布满灰白色坏死点。
    *   **肝坏死 (白点病)：** 肝脏表面布满 **针尖大灰白色坏死灶**（注意与DVH的红色出血斑绝对区分）。
*   **实验室题眼：** 接种鸭成纤维细胞(DEF)形成 **多核巨细胞/合胞体 (Syncytia)**。

### 3. 鸭瘟 (DVE / 鸭病毒性肠炎)
*   **致病原：** 鸭疱疹病毒1型 (DEV，有包膜)。
*   **易感日龄：** 所有日龄，特别是**成年产蛋鸭/种鸭**伤亡惨重。
*   **临床特征题眼：** “大头瘟”（头颈皮下水肿），**流泪怕光，眼睑粘连**睁不开，排绿色稀粪。
*   **解剖特征题眼 (消化道三位一体)：**
    *   食道与泄殖腔：黏膜表面覆盖 **黄白/褐绿色假膜 (固化结痂)**，剥离后留有出血溃疡。
    *   肠道：黏膜淋巴集结处出现 **环形出血/坏死带**。
    *   肝脏：红白相间的 **“靶心状”坏死环**。

### 4. 鸭坦布苏病毒感染 (鸭黄病毒感染)
*   **致病原：** 鸭坦布苏病毒 (黄病毒科)。
*   **易感对象：** 产蛋鸭（爆发性减蛋）。
*   **临床/解剖特征题眼：** 
    *   **“鸭的减蛋综合征”：** 产蛋率几天内从 90% 断崖式跌至 10% 甚至绝产。
    *   **卵巢病变：** 卵泡严重充血、出血、萎缩、破裂（引发卵黄性腹膜炎）。
    *   **神经症状：** 部分病鸭表现为脑炎症状（翻转、共济失调）。

### 5. 小鹅瘟 (GPV)
*   **致病原：** 鹅细小病毒 (GPV，无包膜单链DNA病毒)。
*   **易感日龄：** **20日龄以内** 的雏鹅或雏番鸭。
*   **解剖特征题眼：** 肠道（尤其是空肠和回肠）发生急性纤维素性坏死性肠炎，渗出物凝固形成 **“腊肠样粪栓” (或称脱落的肠管状假膜)**，完全堵塞肠腔。

### 6. 鸭短喙大舌病 (水禽细小病毒变异株)
*   **致病原：** 新型鹅细小病毒 (N-GPV) 或水禽细小病毒。
*   **易感对象：** 雏鸭（多见于白羽肉鸭、樱桃谷鸭）。
*   **外观题眼：** 喙部骨骼发育障碍，导致 **上喙变短，舌头向外吐出 (大舌头外伸)**；生长极度迟缓（侏儒鸭），骨骼脆弱易折断（类似骨质疏松）。

### 7. 鸭星状病毒感染 (DAstV)
*   **致病原：** 鸭星状病毒。
*   **特征题眼：** 
    *   *DAstV-1/2型：* 引起类似鸭肝炎的致命性肝脏病变。
    *   *DAstV-3型 (近年极高频考点)：* 引起雏鸭/雏鹅极其严重的 **内脏痛风和花斑肾**（心肝肾表面覆盖厚层白色尿酸盐沉积），临床极易与传染性支气管炎（IBV）混淆，但在水禽上见痛风，首推星状病毒。

## 二、 细菌性疾病：纤维素性渗出的“双胞胎”对决

在水禽临床和客观题中，**鸭传染性浆膜炎** 和 **鸭大肠杆菌病** 的剖检病变高度重合（都是“包心包肝”），这是最经典的易混淆考点，必须严格通过以下矩阵进行鉴别。

| 鉴别/特征维度 | 鸭传染性浆膜炎 (R. anatipestifer) | 鸭大肠杆菌病 (E. coli) |
| :--- | :--- | :--- |
| **致病原/革兰氏染色** | 鸭疫里默氏杆菌 (**G-**，微需氧，无芽孢) | 致病性大肠埃希菌 (**G-**，兼性厌氧，有鞭毛) |
| **高发日龄与群体** | **1~8 周龄雏鸭/小鸭**最为敏感。成年鸭极少发病。 | 所有日龄均可发病。 |
| **临床神经表现** | 频发特征性神经症状：**阵发性痉挛、头颈震颤、摇头、腿划水样动作**。 | 极少表现典型的神经症状。 |
| **核心解剖病变 (相似度90%)** | 三大纤维素性炎：**心包炎**（心包膜增厚变白）、**肝周炎**（肝表面覆盖一层可剥离的白色纤维素膜）、**气囊炎**。 | 同左（也是包心包肝）。<br>*附加：成年母鸭常表现为腹膜炎、输卵管炎。* |
| **特异性病变鉴别点** | 常伴随**脑膜炎**（导致神经震颤的原因）。 | 常伴发**肠炎**、腹水，成年鸭表现卵黄性腹膜炎（肚子有恶臭黄色液体）。 |
| **流行病学“江湖地位”** | 水禽专属的高发病率细菌病，感染率往往高于大肠杆菌。 | 广泛存在于各品种家禽的条件致病菌。 |
| **麦康凯琼脂培养** | **不生长**。 | 形成典型的**红色菌落**（发酵乳糖）。 |

**顾问考试防坑口诀：** 
水禽出现“包心包肝”：如果题干强调“小鸭、有摇头痉挛等神经症状、麦康凯不生长”，闭眼选**鸭传染性浆膜炎**；如果题干提到“成年母鸭、腹膜炎、麦康凯长红菌落”，选**大肠杆菌病**。
