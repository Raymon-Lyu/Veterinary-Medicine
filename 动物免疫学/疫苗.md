<center><h1>疫苗与免疫防治</h1></center>

## 基础概念
在讲疫苗之前，我们要有一个基本体系，即机体是如何获得免疫力的，主要通过以下两个途径：
#### 被动免疫
- 免疫力不是自己产生的，而是获得现有抗体的“拿来主义”
- 见效快，持续时间短
##### 天然被动免疫
- 来自于母胎胎盘、初乳(IgM，lgG)、卵黄(IgY)的抗体
- 母源抗体是新生儿抵抗环境病原体感染的重要来源，过高会影响疫苗的接种
##### 人工被动免疫
- 通过注射免疫血清、康复动物血清、卵黄获得其中的抗体
- 主要用于紧急治疗或预防
#### 主动免疫
- 通过激活免疫系统产生抗体和相应的记忆细胞
- 存在潜伏期/诱导期，但持续时间长
##### 天然主动免疫
- 感染了病原体后发生免疫应答而获得的免疫抵抗力
##### 人工主动免疫
- 通过接种疫苗而获得对特定抗原的特异性免疫力
## 传统疫苗
传统疫苗主要可以分为两类：活疫苗(弱毒苗)和灭活疫苗，此外还包括由细菌毒素制成的类毒素

| **特性**    | **活疫苗 (弱毒苗)**     | **灭活疫苗 (死苗)**  |
| --------- | ----------------- | -------------- |
| **病原体状态** | 活着，能繁殖            | 死的，不能繁殖        |
| **免疫类型**  | 体液 + 细胞 + 黏膜 (全面) | 主要是体液免疫 (产生抗体) |
| **接种剂量**  | 小                 | 大              |
| **免疫次数**  | 通常1次              | 通常需2-3次        |
| **安全性**   | 有返强/散毒风险          | 安全             |
| **佐剂**    | 一般不需要             | **必须需要**       |
| **保存**    | 困难 (需冷冻)          | 较易 (冷藏)        |

### 活疫苗
- 通常被称为弱毒苗，通过变弱的病原体来促使免疫系统产生免疫力
##### 制备方法
- 传代致弱：让病毒在不适应的宿主里进行繁殖，从而获得低毒力的病原体
	[[RNA病毒#黄病毒科|猪瘟]]兔化弱毒苗
- 异源筛选：找一种“长得像”但毒性低的病毒
	用火鸡疱疹病毒来预防鸡的马立克氏病

##### pros&cons
- 接种的病毒可以模拟免疫应答反应，产生的免疫力强、持续时间久，所需的抗原用量少
- 弱毒苗存在突变的可能，保存也比较困难
### 灭活疫苗
- 保留了免疫原性的抗原物质
##### 制备方法
- 通过物理或化学方法来杀死病原体使其丧失感染性但保留抗原性
	常用的灭火手段包括甲醛、烷化剂
- **与[[#传统疫苗]]的区别**：需要佐剂的协助，激活免疫反应
##### pros&cons
- 安全、易储存、研制方便
- 剂量大、免疫期短、主要引起体液免疫
### 类毒素
细菌致病主要依靠其外毒素，所以通过甲醛去毒后可以制备类毒素，如破伤风类毒素、白喉类毒素
