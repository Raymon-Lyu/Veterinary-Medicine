## 抗细菌免疫
### 细菌感染
细菌致病力的强弱利用**毒力**来描述
评价毒力可以从两个角度出发：
##### 侵袭力
- 指的是病原菌突破宿主的身体防线，在宿主体内定殖、扩散的能力
- **定殖**：主要发生在宿主的黏膜上，细菌通过*抵抗宿主体内粘液分泌或是机械摆动*实现在宿主体内存留并开始繁殖
- **扩散**：细菌通过分泌水解酶类物质来使得组织基质变松散，利于病菌通过，这些酶类包括透明质酸酶(金葡菌、链球菌)、链激酶(溶血性链球菌，激活纤维蛋白酶)、链道酶(链球菌)、蛋白水解酶、胶原酶和弹性蛋白酶
##### 毒素
- 可以分为外毒素和内毒素
###### 外毒素
- 病原细菌在生长过程分泌到胞外的毒素或是细菌溶解后释放
- 化学本质是**蛋白质**
###### 内毒素
- 是$Gram^{-}$的细胞壁物质
- 主要成分是脂多糖(LPS)
- 毒性相对较弱，但可以引起[[第四章 炎症|炎症反应]]
###### 类毒素
- 利用[[#外毒素]]制成，[[#内毒素]]不可以
- 可以用下面的关系来理解：$$\text{类毒素}=\text{外毒素➖毒性➕抗原性}$$

> 类毒素相较于外毒素，其发挥毒素作用的区域被破坏，但是其[[抗原#抗原表位|抗原决定簇]]活性仍存在

###### 总结归纳

|           | 外毒素       | 内毒素             |
| :-------- | :-------- | :-------------- |
| **产生菌**   | 主要是革兰氏阳性菌 | 革兰氏阴性菌          |
| **化学成分**  | 蛋白质       | 脂多糖             |
| **释放时间**  | 一般随时分泌    | 菌体死亡裂解后释放       |
| **致病特异性** | 不同外毒素各不相同 | 不同病原菌的内毒素作用基本相同 |
| **毒性**    | 强         | 弱               |
| **抗原性**   | 完全抗原，抗原性强 | 不完全抗原，抗原性弱      |
| **制成类毒素** | 能         | 不能              |
| **热稳定性**  | 差         | 耐热性强            |

### 抗细菌感染
##### 抗胞外菌感染
- 主要依赖[[适应性免疫应答#体液免疫|体液免疫]]
其抵抗的途径包括：
1. 抗毒素免疫
	抗体中和[[#外毒素]]，被吞噬细胞清理消化
2. 抗体抑制细菌繁殖
	抗体“吸附”在细菌表面可以组织其与机体接触并形成凝集，从而实现抑制细菌生长
3. 抗体和补体对细菌实现溶解
	细菌抗原和IgG、IgM结合后，通过激活[[补体系统#经典途径|经典途径]]形成MAC溶解细菌
4. 抗体调理吞噬
	调理可以促进巨噬细胞的吞噬作用，如C3b结合细菌(非特异性)。抗体的调理是特异性的，吞噬细胞表面的Fc受体可以结合在病原菌上抗体的Fc片段
5. 抗体抑制细菌的吸附
	病原体对细胞发生毒害作用首要是发生黏附，粘膜表面的分泌型IgA可以阻止该过程
##### 抗胞内菌感染
- 主要依赖[[适应性免疫应答#细胞免疫|细胞免疫]]
- 与胞外菌感染过程不同，当菌体存在于胞内时，抗体和吞噬细胞的功能无法正常发挥，因此依赖$T_H1$和`CTL`介导的细胞免疫
## 抗病毒免疫
### 病毒感染
- 病毒感染细胞并不是直接“毒害”细胞，而是改变细胞特性引起免疫系统的“误伤”
病毒主要通过以下机制来实现：
1. 杀细胞效应
	病毒在宿主细胞内增殖引起宿主细胞溶解死亡
2. 细胞膜改变
	某些病毒存在引起在细胞表面表达**病毒融合蛋白**，实现感染细胞与未感染细胞融合成多核巨细胞
3. 细胞转换
	病毒的核酸整合到宿主的基因组中可能会激活原癌基因或是抑制抑癌基因，从而导致细胞癌变
4. 持续性感染
	病毒长期存在宿主机体内，感染免疫细胞，导致机体免疫力降低，引起继发感染
5. 抗病毒反应引起机体细胞的损伤
	[[变态反应]]的Ⅱ、Ⅲ、Ⅳ型均会发生
### 抗病毒感染
##### 非特异性免疫
特异性免疫发生前机体对病毒的抵抗力发生途径，主要依赖：
1. 干扰素
	病毒感染的细胞分泌干扰素诱导周围正常细胞发挥抗病毒途径，如产生分解病毒RNA的酶、抑制蛋白质合成
2. 单核-吞噬细胞系统和NK细胞
	无MHC限制性，不依赖特异性抗体，直接发挥吞噬清理作用
##### 特异性免疫
- **体液免疫**：
	- **抗体中和**：可以分泌抗体来阻止病毒吸附和穿入宿主细胞的过程，相应的抗体可以分为**中和抗体**(粘膜免疫的IgA)和**非中和抗体**，前者可以结合病毒的表面蛋白和刺突，后者结合病毒的碎片，如核衣壳
	- **抗体介导吞噬**：无论是游离的病毒颗粒还是被病毒感染的宿主细胞，抗原结合抗体后，既容易被吞噬细胞清理，又容易激活补体
- **细胞免疫**：
	- `CTL`**介导的抗病毒作用**：被感染细胞通过`MHC-Ⅰ`呈递病毒抗原，CTL引起靶细胞的凋亡
	- $T_H1$细胞释放细胞因子活化吞噬细胞，也可以介导[[变态反应#迟发型过敏反应|迟发型变态反应]]清除胞内病毒
