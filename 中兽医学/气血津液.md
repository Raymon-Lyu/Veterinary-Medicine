# 气血津液
## 气
### 概述
参考[[元气论#理论基础|气的基本概念]]，是构成机体和维持生命活动的基本物质，存在以下特征：
1. 物质性与运动性：气是运动不息的物质，气是构成万物的本原
2. 生命之本：“气聚则生，气散则死”
3. 存在形态：气以两种形态存在，无形如机体的元气、宗气；有形则为各种脏腑器官
### 气的生成
气的生成来源于先天的禀受以及后天的获取：
- **先天之气**：禀受于父母的**先天之精气**，这是动物体生长发育和生殖的根本
- **后天之气**：动物体不仅要从自然界摄取清气，还必须摄入食物才能维持生命。后天之气主要由**肺吸入的自然界清气**和**脾胃运化的水谷精微之气**结合而成，这是维持生命活动的主要物质
### 气的运动
气的运动被称为**气机**
- 气的运动有**升、降、出、入**四种方式，如脾气主升，胃气主降
- 脏腑在气机有不同的分工： 
	- 五脏：心肺在上，其气宜降；肝肾在下，其气宜升；脾胃位于[[六腑#三焦|中焦]]，联通上下，是气体升降的枢纽
	- 六腑：以通为用，宜降
- 气的运动依赖**血、津液**等载体
### 气的生理功能
气在机体内会发挥强大且多维的作用：
1. 推动作用
2. 温煦作用
3. 防御作用
4. 固摄作用
5. 气化作用
6. 营养作用
