# 抗体-药物偶联物(ADC)的纯化
## ADC的基本性质
### 什么是ADC
抗体偶联药物，由靶向肿瘤特异性抗原或者靶向肿瘤相关抗原的单克隆抗体，与不同数目的小分子细胞毒素通过连接子偶联而成的

总体为三部分的要求
- **抗体**：特异性、稳定性、内化
- **连接方式与连接子**：稳定性、可控释放、亲水性
- **药物**：高效力、稳定性、潜在旁观者效应。

## 典型的制造工艺
mAb的制取：
从培养液中分离出细胞和碎片，获得含有药物的清液 → 第一次纯化，特异性地“捕获”目标药物 → 第二步纯化，去除更多杂质 → 最后一步纯化，也叫“抛光”，达到极高纯度 → 将药物溶液浓缩并置换成最终的制剂缓冲液 → 加入稳定剂等辅料，并进行除菌过滤 → 最终得到可用于灌装的高纯度药物

### 分离需解决的问题
- **分离不同药物抗体比（DAR）的组分 (Separation of DAR Species)**：生产得到的ADC通常是包含不同数量药物分子的抗体混合物（例如，DAR=0, 2, 4, 6, 8的混合物）。这个DAR值是影响药物疗效和安全性的关键质量属性 。DAR过高的组分可能导致聚集和体内清除过快，影响安全性和药效；而DAR过低的组分则药效不足 。因此，需要通过层析手段（如疏水作用层析 HIC）分离这些组分，得到更均一的ADC产品。
- **去除聚集体 (Removal of Aggregates)**：在ADC的生产和纯化过程中，蛋白质可能会形成聚集体 。这些聚集体是一个主要的制造难题，因为它会引发免疫原性，可能导致严重的不良反应。
- **清除游离的药物-连接子及相关杂质 (Removal of Free Drug-Linkers)**：ADC所携带的小分子“有效载荷”（payload）通常具有极高的细胞毒性 。因此，在偶联反应后，任何残留的、未结合到抗体上的游离药物-连接子都必须被彻底清除，以确保最终药品的安全性 。由于一些药物-连接子疏水性强，容易自我缔合形成胶束，这给它们的清除带来了挑战，有时单纯的TFF方法不足以完全去除，需要额外的层析步骤或优化工艺 。

## 目前几种的分离方法
| 纯化方法                            | 优点                                                                                                                       | **缺点/局限性**                                                                                     |
| :------------------------------ | :----------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------- |
| **切向流过滤 (TFF) / 超滤-渗滤 (UF/DF)** | - 应用广泛，可用于去除有机溶剂、偶联相关杂质以及进行缓冲液交换。<br>- 是一种简单且稳健的纯化方法。<br>- ADC的回收率高，通常大于90%。<br>- 对小分子杂质的清除率受pH、温度、膜载量等工艺参数影响最小。         | - 清除疏水性强的细胞毒性药物-连接子时可能具有挑战性，因为它们可能自缔合形成胶束。<br>- 无法去除聚集体。<br>- 无法分离不同药物抗体比（DAR）的组分，最终产物是DAR混合物。 |
| **体积排阻层析 (SEC)**                | - 主要目标是去除聚集体，这对降低免疫原性至关重要。<br>- 也可用于缓冲液交换和去除小分子杂质。<br>- 操作条件温和，通常在室温和中性pH缓冲液中进行。<br>- 是实验室规模ADC纯化的金标准方法。                 | - 由于其分辨率较低，不适合用于分离不同DAR的组分。<br>- SEC固定相与ADC之间可能存在非特异性的疏水或静电相互作用。<br>- 在大规模生产工艺中，通常首选TFF。       |
| **疏水作用层析 (HIC)**                | - 可以在非变性（天然）条件下进行，无需预处理即可分离ADC。<br>- 能够有效分离不同DAR值的组分（如DAR=0, 2, 4, 8）。<br>- 可用于完全去除DAR=0的未偶联抗体，这可能有助于提高药物疗效。             | - 纯化工艺仍需进一步优化，例如提高峰分辨率和ADC回收率。<br>- 某些常用盐（如硫酸铵）可能导致ADC回收率降低。                                   |
| **离子交换层析 (IEC)**                | - 是一种有前景的ADC纯化方法。<br>- 通常具有高回收率。<br>- 能够去除包括小分子和内毒素在内的非抗体相关化合物。                                                          | - 目前将其用于ADC分析或纯化的研究报告数量有限。                                                                     |
| **羟基磷灰石层析 (HA)**                | - 是一种有用的单克隆抗体纯化方法。<br>- 能显著减少抗体聚集（例如从60%降至0.1%）。<br>- 可有效去除DNA、内毒素以及Protein A层析后残留的杂质。                                   | - 目前将HA层析应用于ADC纯化的报道很少。                                                                        |
| **膜层析 (MC)**                    | - 是基于树脂的层析的一个有前景的替代方案。<br>- 能够同时实现去除聚集体、残留药物-连接子以及分离DAR组分。<br>- 可在高流速和低压力下操作。<br>- 避免了传统层析需要高浓度盐和长接触时间的问题，从而减少了ADC聚集的风险。 | - 已发表的关于其在ADC纯化中应用的文献还比较少。                                                                     |
### ADC纯化方法详解
下面将基于《Journal of Separation Science》的综述文章，详细介绍在抗体药物偶联物（ADC）生产中三种关键的纯化技术：切向流过滤（TFF）、疏水作用色谱（HIC）和膜色谱法（MC）。
#### 1. 切向流过滤 (Tangential Flow Filtration, TFF)
##### 具体操作
切向流过滤，是生物制药下游工艺中的一个基础单元操作，尤其在ADC生产流程中扮演着不可或缺的角色。它主要用于基于分子大小差异的“粗”分离，而非需要高分辨率的精细分离。
参考的文献使用的TFF分离纯化在一套实验室规模的系统上进行，其核心设备与材料如下：
- **TFF系统**: 采用了一套台式UF/DF系统（依据论文图1配置），包含蠕动泵、带搅拌的料液罐（反应器）、压力传感器（P1, P2, P3）、渗透液收集罐以及连接管路。
<img src='https://aiche.onlinelibrary.wiley.com/cms/asset/05df4044-a37a-4f55-8933-03c63622fe87/btpr2923-fig-0001-m.jpg' style="display: block; margin-left: auto; margin-right: auto; width: 75%;">
(*上图展示了典型的实验室规模TFF（UF/DF）系统设置示意图。料液在反应器中通过泵驱动，在膜表面形成切向流。渗透液穿过膜被收集，同时渗滤缓冲液被加入反应器以维持体积恒定。*)
- **TFF膜包**: 核心分离元件为**Millipore Pellicon-3 Ultracel（再生纤维素）膜包**。该膜包的截留分子量（MWCO）为**30 kDa**，膜面积为88 cm²，采用C型筛网。
- **抗体与缓冲液**:
    - 使用了两种不同的单克隆抗体（mAb）：34C3和AB095。
    - 渗滤（DF）缓冲液为15 mM组氨酸缓冲液（pH 6.0）。
- **目标分离物（小分子杂质）**: 实验中向抗体溶液中加入了五种代表性的工艺杂质：
    - **溶剂**: 二甲基亚砜 (DMSO) 和 N,N-二甲基乙酰胺 (DMA)。
    - **反应相关物**: 氧化三(2-羧乙基)膦 (TCEP=O)、乙二胺四乙酸 (EDTA) 和 N-乙酰-L-半胱氨酸 (NAC)。
整个分离纯化过程严格按照以下步骤进行：
1. **系统准备**: 每次运行前，使用0.1 N的氢氧化钠对整个TFF系统进行彻底的清洗和消毒，随后用纯水冲洗，并使用渗滤缓冲液进行平衡。
2. **上样**: 将含有特定浓度小分子杂质的抗体溶液（约50 mL）加入到TFF系统的反应器中，初始的蛋白膜载量为113 g/m²。
3. **恒定体积渗滤 (Diafiltration)**:
    - 启动系统，在**恒定体积、恒定跨膜压差**的模式下运行。
    - 泵驱动料液在膜表面高速切向流动，标准操作条件设定为：
        - **切向流速**: 300 L·m⁻²·hr⁻¹ (即 44 mL/min)
        - **跨膜压差 (TMP)**: 18-20 psi
        - **温度**: 室温 (约 21°C)
    - 在操作过程中，渗透液不断穿过膜被收集，同时系统会自动补充新鲜的渗滤缓冲液，以精确维持反应器中料液体积不变。整个过程通常进行8至10个渗滤体积（Diavolumes, DV）。
4. **取样**: 在第1、2、4、6、8等不同的渗滤体积点，分别从**截留液**和**渗透液**出口采集1 mL样品用于后续浓度分析。
5. **浓度分析**: 使用经过验证的分析方法（如HPLC和LC-MS）对采集样品中五种小分子杂质的精确浓度进行定量检测。
##### 结果评价与分析
对TFF分离效果的评价主要基于对杂质清除效率的量化和对工艺稳健性的考察。
- **主要结果：杂质清除效率极高** 实验结果（论文图2）显示，所有五种小分子杂质的浓度均随渗滤体积数的增加呈现出完美的线性下降。计算得出的**筛分系数（S）在0.85至0.99之间**，非常接近理论最大值1。这表明，该TFF工艺能够以近乎理想的状态高效清除ADC工艺中常见的各类小分子杂质。例如，对于初始浓度约10%的DMA和DMSO，经过8个渗滤体积的操作后，其残留浓度降低了超过2000倍。
<img src='https://aiche.onlinelibrary.wiley.com/cms/asset/abe1e9b9-0f9a-4f71-8d6f-f299da4c9a42/btpr2923-fig-0002-m.jpg'>
尽管TFF是一个非常稳健的工艺，但它在处理一些疏水性极强的游离药物-连接子时会面临挑战。这类分子倾向于在水溶液中自我缔合形成胶束，导致其表观分子尺寸增大，甚至可能超过膜的截留分子量，从而难以被有效清除。研究表明，通过在反应后加入特定的“淬灭剂”（如L-半胱氨酸），可以改变这些残留物的化学性质，降低其疏水性，从而显著提高其在TFF过程中的清除效率。

#### 2. 疏水作用色谱 (Hydrophobic Interaction Chromatography, HIC)
疏水作用色谱是解决ADC产品“异质性”问题的核心技术，特别是在分离具有不同药物抗体比（DAR）的组分方面，其作用至关重要。ADC的异质性是其安全性和有效性的关键质量属性，因此控制和分离这些组分是工艺开发的核心目标之一。
HIC的分离原理基于蛋白质分子表面疏水性的差异。抗体本身具有一定的疏水性，当其通过化学连接子偶联上疏水性的小分子细胞毒性药物后，其整体疏水性会显著增加。并且，偶联上的药物分子越多（即DAR值越高），ADC分子的疏水性就越强。HIC正是利用了这一特性，具体过程如下：
1. **结合**：将ADC样品置于高浓度的盐溶液（文献中采用的是$NaCl$）中，这种高盐环境会增强蛋白质与HIC疏水性固定相（文献中提到采用的是苯基树脂）之间的相互作用，使得不同DAR值的ADC组分都结合到色谱柱上。
2. **洗脱**：通过一个逐渐降低盐浓度的梯度，分子间的疏水相互作用会减弱。疏水性最弱的组分，即未偶联的裸抗（DAR=0），会最先被洗脱下来。随着盐浓度的进一步降低，DAR=2, DAR=4, DAR=6, DAR=8的组分会因其不断增强的疏水性而依次被洗脱，从而实现对不同DAR物种的高分辨率分离。
文献中的下图反映了整个洗脱过程
<img src='https://www.mdpi.com/antibodies/antibodies-09-00016/article_deploy/html/images/antibodies-09-00016-g004-550.jpg' style="display: block; margin-left: auto; margin-right: auto; width: 75%;">
- 第一个洗脱峰(DAR=0)：在约6.7 mL处出现。这个峰的位置非常靠前，这代表了未偶联的裸抗体，其疏水性最弱，在高盐的结合缓冲液条件下也无法与疏水填料发生有效结合。图中的橙色线（电导率）在此处处于最高值，也印证了这是在高盐的上样阶段流出的组分。
- 第二个洗脱峰(DAR=1)：在约44.4 mL处洗脱下来。说明每个抗体只偶联一个药物分子的ADC的疏水性强于裸抗，因此能够结合在层析柱上。随着洗脱梯度（绿色线，代表含有机溶剂的B缓冲液浓度）的进行，疏水相互作用减弱，DAR=1的组分被洗脱下来。
- 第三个洗脱峰(DAR=2)：在约55.7 mL处洗脱下来。这是偶联了两个药物分子的ADC，是三个组分中疏水性最强的。因此，它与层析柱的结合力也最强，需要更高浓度的有机溶剂（梯度后期）才能被洗脱。
橙色曲线（电导率曲线）为什么先下降再上升呢？
	这并不是梯度洗脱的一部分，而是一个典型的**进样伪峰**或**溶剂峰**。当您将样品（溶解在自己的缓冲液中）注入到正在流动的初始高盐缓冲液（流动相A）中时，样品本身的溶剂环境和流动相的溶剂环境有微小差异（比如pH、盐种类、添加剂等），这会引起电导率检测器的瞬间扰动，从而产生一个快速的波动。这标志着您的样品团已经进入了色谱柱。

#### 3. 膜色谱法 (Membrane Chromatography, MC)

- 详细介绍与应用
膜色谱法是一种结合了膜分离和色谱法优点的新型纯化技术，被视为传统颗粒填料柱层析的一个高效替代方案。它将带有功能基团（如离子交换基团、疏水基团）的微孔滤膜作为固定相，与传统色谱柱相比，其结构和流体动力学特性有着根本的不同。
其核心优势在于传质机制。在传统的多孔颗粒填料色谱柱中，溶质分子需要通过缓慢的扩散作用进入颗粒内部的孔道才能与结合位点相互作用，这个过程是限速步骤。而在膜色谱中，料液直接通过对流的方式流过多孔膜的通道，溶质分子可以快速、直接地接触到固定在通道表面的功能基团。这种以对流为主的传质方式，使得膜色谱可以在极高的流速下运行而不会显著增加背压或降低分离度。
在ADC纯化领域，膜色谱展现了巨大的应用潜力，特别是在开发集成、连续的纯化工艺方面：
- **集成化一步纯化**：膜色谱的“即用型”和易于串联的特性使其非常适合构建多模式纯化系统。例如，研究表明，可以将一片阳离子交换膜（用于结合带正电的ADC并去除带负电的杂质）与一片疏水相互作用膜（用于分离不同DAR组分）直接串联起来。这样的“串联膜色谱”系统，可以在一个单元操作中，一次性完成对ADC粗品中**聚集体、游离药物-连接子以及不同DAR组分**的分离和去除，极大地简化了工艺流程、缩短了处理时间。
- **处理高难度ADC**：对于一些有效载荷（payload）疏水性极强、毒性极高（如PBD类分子）的ADC，其纯化过程更具挑战性。膜色谱同样在这种场景下显示出良好的纯化效果，能够有效去除这些高风险的残留杂质。
总而言之，膜色谱法凭借其高速度、高效率和高灵活性的特点，为开发更经济、更紧凑、更高效的下一代ADC纯化平台提供了强有力的技术选择。
