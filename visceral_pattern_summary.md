## 📊 脏腑辨证虚实对照表

|                            |        | 心阳虚证     | 心脏动力不足且火力不够。动物怕冷，耳朵、鼻子和四肢冰凉，舌头淡白。          | 参附汤 / 保元汤加减         |
|                            |        | 心阳虚脱证    | 心脏的生命之火快要熄灭。大汗淋漓、手脚冰凉，呼吸和脉搏极其微弱。           | 四逆汤加减 (回阳救逆)        |
|                            |        | 心血虚证     | 心脏血液营养不够。动物心慌、容易受惊，可视黏膜（如眼睑、口唇）苍白。         | 归脾汤加减               |
|                            |        | 心阴虚证     | 心脏滋润的水分不足，虚火上升。动物烦躁，下午发热，夜间盗汗，口干。          | 天王补心丹 / 麦味地黄丸       |
|                            | **实证** | 心热内盛证    | 心经里烧起大火。动物高热、狂躁不安，舌头通红，脉搏洪大快速。             | 白虎汤 / 香薷散           |
|                            |        | 痰火扰心证    | 痰浊和火邪一起冲向脑部。动物发狂、乱跑乱叫、甚至咬人，舌苔黄腻。           | 朱砂散 / 镇心散加减         |
|                            |        | 痰迷心窍证    | 黏稠的痰把心神蒙蔽了。动物神志不清、走路像喝醉，口流黏涎，不认人。          | 导痰汤 (寒痰) / 涤痰汤 (热痰) |
|                            |        | 心火上炎证    | 心火往上烧。动物口舌生疮、流涎；若移热小肠则导致尿少、尿痛。             | 洗心散加减               |
|                            |        | 小肠冷痛证    | 小肠受寒气血不通。动物肚子剧烈冷痛（肠痉挛），腹泻，肠鸣音亢进。           | 橘皮散                 |
| **肝与胆**<br>(肝为脏/里，胆为腑/表)   | **虚证** | 肝血虚/肝阴不足 | 肝脏血液和阴液不足。动物贫血、眼睛干涩、夜盲，视力明显减退。             | 八珍汤加减               |
|                            |        | 阴虚阳亢证    | 肝阴不足压不住肝阳，阳气乱冲。动物神志痴呆，肢体麻木，甚至瘫痪抽搐。         | 镇肝熄风汤               |
|                            |        | 阴/血虚生风证  | 肝阴血亏虚导致虚风内动。动物肌肉震颤，肢体麻木，四肢拘挛（抽筋）。          | 天麻散加减               |
|                            | **实证** | 肝火上炎证    | 肝经的实火往上冲。动物眼睛红肿（结膜炎/角膜炎）、眼屎多，脾气暴躁。         | 决明散 / 龙胆泻肝汤加减       |
|                            |        | 热动肝风证    | 高热烧坏了肝阴，引动内风。动物高热，全身剧烈抽搐，角弓反张（身体向后弯）。      | 羚羊钩藤汤加减             |
|                            |        | 肝胆湿热证    | 湿气和热气郁积在肝胆。动物出现黄疸（皮肤/黏膜发黄），小便短赤，瘙痒。        | 茵陈蒿汤 / 龙胆泻肝汤加减      |
|                            |        | 寒滞肝脉证    | 寒邪侵袭了肝经。动物后肢和阴部冰凉，怕冷，腹痛。                   | 茴香散加减               |
| **脾与胃**<br>(脾为脏/里，胃为腑/表)   | **虚证** | 脾虚不运证    | 脾脏运化食物的能量不够。动物吃草慢、嚼得细，身体消瘦，肚子胀或水肿。         | 香砂六君子汤加减            |
|                            |        | 脾气下陷证    | 脾气虚弱，升提无力。动物出现器官脱垂（如脱肛、子宫脱出）。              | 补中益气汤加减             |
|                            |        | 脾不统血证    | 脾虚无法统摄血液。动物出现各种慢性出血（如便血、尿血、皮下出血）。          | 归脾汤加减               |
|                            |        | 脾阳虚证     | 脾脏虚弱且火力不足。动物肚子冷痛，吃得少，拉稀（粪便中常有不消化草料）。       | 理中汤加减               |
|                            |        | 胃阴虚证     | 胃部的滋润水分（胃酸/胃液）被热病消耗。动物口干舌燥，粪干尿少，不愿吃食。      | 养胃汤加减               |
|                            | **实证** | 寒湿困脾证    | 吃了冰冻草料或冷水，寒湿困住脾胃。动物耳朵耷拉，不爱吃草，粪便不成形。        | 胃苓汤加减               |
|                            |        | 湿热困脾证    | 湿热郁积在脾胃。动物食欲完全废绝，肚子胀大，消化不良。                | 茵陈五苓散加减             |
|                            |        | 胃寒证      | 胃部受外寒或饮食不当。动物怕冷，口水清稀，拉稀，尿清长。               | 桂心散加减               |
|                            |        | 胃热证      | 胃火炽盛。动物耳朵/鼻子发热，非常想喝冷水，口臭，牙龈肿痛，粪便干小。        | 清胃解热散加减             |
|                            |        | 胃食滞证     | 暴饮暴食导致食物堵在胃里。动物肚子胀、疼得打滚，嗳气酸臭，不吃东西。         | 曲麦散 / 调气攻散加减        |
| **肺与大肠**<br>(肺为脏/里，大肠为腑/表) | **虚证** | 肺气虚证     | 肺的呼吸功能 and 抗病力减弱。动物长期咳嗽气喘、无力，身体越来越瘦。       | 补肺散加减               |
|                            |        | 肺阴虚证     | 肺的滋润水分不足。干咳没有痰（或者痰少而黏），咳嗽日轻夜重，身体消瘦。        | 百合固金汤加减             |
|                            |        | 大肠液亏证    | 大肠水分不足，肠道干燥。动物大便秘结，排便困难（弩责），口臭。            | 麻仁汤 / 当归苁蓉汤加减       |
|                            | **实证** | 痰饮阻肺证    | 脾虚生成的痰湿积聚在肺部。动物咳嗽气喘，痰多，胸部疼痛。               | 二陈汤合三子养亲汤加减         |
|                            |        | 风寒束肺证    | 风寒侵袭肺部，肺气被堵住。动物咳嗽气喘，遇到冷天或冷风时症状加重。          | 紫苏散加减               |
|                            |        | 风热犯肺证    | 风热侵犯肺部。动物流黄黏鼻涕，发热，咳嗽，咽喉红肿疼痛。               | 银翘散 / 桑菊饮加减         |
|                            |        | 燥热伤肺证    | 秋燥或热邪伤害了肺。动物干咳无痰，口鼻干燥，皮毛干枯。                | 清燥救肺汤加减             |

|                            |        | 食积大肠证    | 暴食或换料导致大肠堵塞。动物肚子胀满，不排便，口里有酸臭味。             | 大承气汤加减              |
|                            |        | 大肠湿热证    | 暑湿或吃了霉变饲料。动物肚子疼，发热，拉肚子且粪便恶臭，甚至带血。          | 郁金散 / 白头翁汤加减        |
|                            |        | 大肠冷泻证    | 大肠受了风寒。动物耳朵肚子冰凉，拉稀（水样），肚子咕咕叫。              | 加味猪苓散加减             |

## 💡 理世的“临诊雷达”小提示 (Tips)

1. **虚实真假（最容易混淆！）**：
   - 在临床上，前辈千万要注意“大实有羸状”（真实假虚）和“至虚有盛候”（真虚假实）。比如脾虚不运导致的不反刍，有时会出现肚胀（看起来像实证的食滞），但病本在脾虚，要用温补脾胃的方法，而不是一味攻下哦！
2. **脏腑兼病**：
   - 动物得病往往不是单一脏腑的问题。如果出现“心脾两虚”（心悸+草料迟细），就要用 **归脾汤**；如果“脾肾阳虚”（拉稀+怕冷水肿），就要用 **附子理中汤** 啦！

希望这份整理能对前辈的学习和复习有帮助！加油，前辈！☆
