# 🐾 中兽医特色诊法：察口色与切脉复习笔记

> 📌 **复习导读**：在中兽医学的“四诊”（望、闻、问、切）中，**察口色**（望诊特色）和**切脉**（切诊特色）是最具代表性的中兽医诊断方法。本笔记已优化为 **Obsidian 双向链接与 Callout 语法格式**，方便随时导入知识库和复习。

---

## 👅 第一部分：察口色（望诊特色）

口色是畜体气血的外荣。口色变化能直观反映体内**气血盈亏**与**脏腑虚实**。

### 1. 察色部位与诊断分属
不同动物在观察口色时侧重点不同，口腔各部位亦分属五脏：

| 动物类别 | 主要察色部位 | 诊断要点与技巧 |
| :--- | :--- | :--- |
| **马属动物** | 唇、舌、卧蚕、排齿 | 需牵鼻绳，翻唇看齿，手掌竖立入口看舌 |
| **牛与羊** | **舌底**（最重要）、卧蚕、仰池、颊部 | 卧蚕位于舌下，左为金关，右为玉户。仰池为卧蚕周围陷窝 |
| **猪** | 舌 | 观察舌质与舌苔颜色 |

> [!NOTE] **口腔部位与五脏分属口诀**
> * **舌色应心**（舌 ➡️ 心）
> * **唇色应脾**（唇 ➡️ 脾）
> * **金关应肝**（左卧蚕 ➡️ 肝）
> * **玉户应肺**（右卧蚕 ➡️ 肺）
> * **排齿应肾**（排齿 ➡️ 肾）
> * **口角应三焦**（口角 ➡️ 三焦）

### 2. 生理常色（健康状态）
* **四季气候影响**：`“春如桃花夏似血，秋如莲花冬似雪”`
* **动物与年龄差异**：猪口色偏红，马骡次之，反刍兽（牛羊）偏淡；幼畜偏红，老龄动物偏淡。

### 3. 五种病理口色及临床意义

| 口色 | 临床主病 | 常见具体表现与兽医临床指征 |
| :---: | :--- | :--- |
| **白色** | **虚证、气血不足** | 营血不能充盈，或气虚运血无力。<br>• **淡白**：血虚。常见于长期脾胃虚弱引起的营养不良、慢性贫血、虫积。<br>• **苍白（淡白灰暗）**：气血极度虚弱。多见于剧烈内脏出血或重度虫积。<br>• **淡白带青**：虚寒。多见于脾胃虚寒、慢草（不反刍）。 |
| **赤色** | **热证、气血沸涌** | 气血得热则行，血管充盈。<br>• **赤红/鲜红**：表热、卫分或气分阶段。见于热性传染病早中期。<br>• **赤紫/深绛**：热入营血、气滞血瘀。见于传染病后期、喘气病、肠扭转、结症后期。<br>• **局部特征**：舌尖红（心火上炎）；舌边红（肝胆有热）；舌红无苔（阴虚火旺）；深红瘀点（热毒炽盛，发斑先兆）。 |
| **青色** | **寒证、痛证、风证** | 感受寒邪或疼痛，阳气受损，经脉瘀阻。<br>• **青白**：脏腑虚寒。常见于胃寒、脾寒、外感风寒及寒冷腹痛。<br>• **青黄**：寒湿内挟。<br>• **青紫**：寒极、肝风内动或气血瘀滞。<br>• **青紫干燥晦暗**：气滞血瘀或心力衰竭危候。 |
| **黄色** | **湿证、黄疸** | 多因肝、胆、脾受湿热交蒸所致。<br>• **阳黄（黄色鲜明）**：湿热黄疸。常见于急性肝炎、胆管阻塞或血液原虫病。<br>• **阴黄（黄色晦暗）**：寒湿黄疸。常见于慢性肝炎或寒湿困脾证。 |
| **黑色** | **寒深、热极** | 病情重危，气血运行极度不畅。<br>• **黑而有津**：阴寒内盛，经脉拘急，气滞血瘀。<br>• **黑而无津**：阳热炽盛，耗津伤气，血热搏结。 |
| **绝色** | **濒死危候** | 见于危重病后期或濒死期。口色呈现**青黑色**或**紫黑色**。 |

### 4. 舌苔与舌形辨证

| 诊察对象 | 分类 | 临床病理意义 |
| :--- | :--- | :--- |
| **舌苔 (胃气熏蒸)** | **薄白苔** | 主表证、寒证。正常健康动物亦为薄白苔（有胃气）。 |
| | **黄苔** | 主热证、里证。越黄越深说明里热越重，由白转黄代表病情加深。 |
| | **灰黑苔** | 主热极、寒湿或虚寒证。 |
| | **苔质 (厚薄/润燥/腐腻)**| • **薄/厚**：辨病位深浅（薄主表/轻，厚主里/重）。<br>• **润/燥**：辨津液存亡（润代表津液未伤，燥代表热盛伤津）。<br>• **腐/腻**：辨宿食与湿浊（似豆渣易刮去属胃气尚好，不易刮去属湿浊内停）。 |
| **舌形 (气血盛衰)** | **淡白胖嫩** | 主**脾肾阳虚**。 |
| | **赤红肿胀** | 主**热毒亢盛**。 |
| | **淡白瘦薄** | 主**气血不足**。 |
| | **伸缩无力** | 主**气血俱虚**。 |
| | **舌斜一侧** | 主**风证**（如中风、脑部病变）。 |

## 脉 第二部分：切脉/诊脉（切诊特色）

中兽医通过手指切按特定动脉，根据脉象的**位、率、形、势**推断病情。

### 1. 不同动物的切脉部位
中兽医根据不同动物的解剖特点，选择最表浅易触的动脉定位：

* **马属动物**：**颌外动脉**（下颌角支内侧）或**双凫脉**（颈总动脉，颈静脉沟下 1/3 波动最明显处）。
* **牛**：**尾中动脉**（尾根腹面）或**颈总动脉**。
* **猪、羊、犬**：**股内动脉**（大腿内侧股沟处）。

### 2. 正常脉象：【平脉】
* **标准**：不浮不沉、不快不慢、至数一定、节律均匀、中和有力、连绵不断。
* **季节偏向**：春偏弦、夏偏洪、秋偏毛（浮）、冬偏石（沉）。
* **一息至数（呼吸与脉搏比例，TCVM 特色标准）**：
  * **马**：一息（一呼一吸）脉跳 **3 次**。
  * **牛**：一息脉跳 **4 次**。
  * **猪、羊**：一息脉跳 **5 ~ 6 次**。

### 3. 双凫脉分脏腑之说（寸关尺定位）
马属动物切双凫脉时，三指排布对应的脏腑如下：

```
【左手三部】                  【右手三关】
   寸 (上) ➡️ 心与小肠            风 (上) ➡️ 肺和大肠
   关 (中) ➡️ 肝与胆              气 (中) ➡️ 脾与胃
   尺 (下) ➡️ 肾与膀胱            命 (下) ➡️ 命门与三焦
```

### 4. 常见病理脉象与临床主病

> [!IMPORTANT] **四大纲脉（最基础的基础脉象）**
> 1. **浮脉**（轻按即得，重按稍减）：**主表证**。浮而有力主表实，浮而无力主表虚。
> 2. **沉脉**（重按始得，如触水中石）：**主里证**。沉而有力主里实，沉而无力主里虚。
> 3. **迟脉**（至数减少，马一息<3次，牛<4次）：**主寒证**。迟而有力主寒实，迟而无力主虚寒。
> 4. **数脉**（至数增多，马一息>=4次）：**主热证**。数而有力主实热，数而无力主虚热。

#### 其他重要病理脉象：
* **虚脉**（三部均无力，按之虚软）：**主虚证**，多为气血两虚。
* **实脉**（三部均充实有力）：**主实证**，见于高热、便结、狂躁、瘀血。
* **滑脉**（往来流利，如盘走珠）：**主痰湿、宿食、实热**。（孕畜滑脉为正常生理现象，非病态）。
* **涩脉**（往来艰涩，如轻刀刮竹）：**主精伤、血少、气滞血瘀**。
* **洪脉**（来盛去衰，脉阔波动大，如波涛汹涌）：**主热盛**（气分大热）。
* **细脉**（脉幅细小，波动小，脉窄）：**主虚证（阴血虚为主）**、湿邪困阻。
* **弦脉**（脉波长，指下挺然，如按琴弦）：**主风证、肝病、痛证、痰饮**。
* **濡脉**（浮细而软，如棉在水）：**主虚证、湿证**。
* **促/结/代脉 (节律异常脉)**：
  * **促脉**：脉来急数，时而一止，止无定数（数而有间歇）。
  * **结脉**：脉来迟缓，时而一止，止无定数（迟而有间歇）。
  * **代脉**：脉来动而中止，不能自还，良久复动，止有定数（有规律的间歇，主气衰危重）。
* **易脉（怪脉/绝脉）**：脉形大小不等，节律全无。如雀啄、屋漏、虾游等，预示生机已绝。
