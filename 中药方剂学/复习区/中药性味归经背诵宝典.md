## <span id="解表药">解表药</span>

> **分类概念**：以发散表邪，解除表证为主要功效的药物。多具**辛**味，主归**肺、膀胱**经。辛能发散，有发汗、解肌作用。

### 📍 辛温解表药
*针对**风寒表证**。药性辛温，具有发散风寒的功效，发汗作用较强。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **麻黄** | 辛、微苦，温 | 肺、膀胱经 | 发汗解表，宣肺平喘，利水消肿（发汗要药） |
| **桂枝** | 辛、甘，温 | 心、肺、膀胱经 | 发汗解肌，温通经脉，助阳化气（有汗无汗皆宜） |
| **细辛** | 辛，温 | 肺、肾、心经 | 祛风散寒，通窍止痛，温肺化饮（助阳解表） |
| **荆芥** | 辛，微温 | 肺、肝经 | 发表散风，透疹消疮，炒炭止血（药性缓和） |
| **防风** | 辛、甘，微温 | 膀胱、肝、脾经 | 祛风解表，胜湿止痛，止痉（散风止痛要药，甘温不燥） |
| **紫苏** | 辛，温 | 肺、脾经 | 发表散寒，行气宽中，解鱼蟹毒（苏叶发表、苏梗安胎、苏子平喘） |
| **白芷** | 辛，温 | 肺、胃、大肠经 | 解表散风，通窍止痛，消肿排脓，燥湿止带（阳明经头痛要药） |
| **辛夷** | 辛，温 | 肺、胃经 | 发散风寒，宣通鼻窍（治鼻渊之要药） |
| **苍耳子** | 辛、苦，温；有毒 | 肺经 | 散风除湿，通窍止痛，杀虫止痒 |
| **生姜** | 辛，温 | 肺、脾、胃经 | 解表散寒，温中止呕，温肺止咳，解半夏、南星毒 |
| **葱白** | 辛，温 | 肺、胃经 | 发表散寒，通阳发汗 |

### 📍 辛凉解表药
*针对**风热表证**。药性多为辛凉，具有疏风散热的功效，发汗作用较缓和。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **薄荷** | 辛，凉 | 肺、肝经 | 疏散风热，清利头目，利咽透疹，疏肝行气（疏散风热要药） |
| **牛蒡子** | 辛、苦，寒 | 肺、胃经 | 疏散风热，宣肺祛痰，利咽透疹，解毒消肿（牛蒡子独降） |
| **蝉蜕** | 甘，寒 | 肺、肝经 | 疏散风热，利咽开音，透疹止痒，明目退翳，息风止痉 |
| **桑叶** | 甘、苦，寒 | 肺、肝经 | 疏散风热，清肺润燥，清肝明目 |
| **菊花** | 甘、苦，微寒 | 肺、肝经 | 疏散风热，平肝明目，清热解毒（清热解毒力胜桑叶） |
| **柴胡** | 苦、辛，微寒 | 肝、胆、肺经 | 和解退热，疏肝解郁，升举阳气（少阳经及肝郁要药） |
| **升麻** | 辛、微甘，微寒 | 肺、脾、胃、大肠经 | 发表透疹，清热解毒，升阳举陷（升阳透疹要药） |
| **葛根** | 甘、辛，凉 | 脾、胃、肺经 | 解肌退热，透疹，生津止渴，升阳止泻 |
| **蔓荆子** | 辛、苦，微寒 | 膀胱、胃、肝经 | 疏散风热，清利头目（治头痛要药） |

## <span id="清热药">清热药</span>

> **分类概念**：以清解里热为主要作用，治疗里热证的药物。药性多**苦寒**，主归**心、肝、肺、胃**等经。

### 📍 清热泻火药
*清除气分实热，主治气分实热证（大热、大渴、大汗、脉洪大）。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **石膏** | 辛、甘，大寒 | 肺、胃经 | 清热泻火，除烦止渴，煅用收湿敛疮 |
| **知母** | 苦、甘，寒 | 肺、胃、肾经 | 清热泻火，滋阴润燥（治气分实热兼滋阴） |
| **栀子** | 苦，寒 | 心、肺、三焦经 | 泻火除烦，清热利湿，凉血解毒（通泻三焦之火） |
| **芦根** | 甘，寒 | 肺、胃经 | 清热生津，川胃止呕，利尿 |
| **竹叶** | 甘、辛，寒 | 心、肺、小肠经 | 清热除烦，生津，利尿（清心火导热下行） |
| **夏枯草** | 苦、辛，寒 | 肝、胆经 | 清肝泻火，解毒消肿，散结（目赤、瘿瘤要药） |

### 📍 清热燥湿药
*苦寒燥湿，治疗湿热证。如湿热泄泻、黄疸、湿疹等。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **黄连** | 苦，寒 | 心、脾、胃、肝、胆、大肠经 | 清热燥湿，泻火解毒（尤擅清心火、胃火、大肠湿热） |
| **黄芩** | 苦，寒 | 肺、胆、胃、大肠经 | 清热燥湿，泻火解毒，止血，安胎（尤擅清肺与上焦热） |
| **黄柏** | 苦，寒 | 肾、膀胱、大肠经 | 清热燥湿，泻火除蒸，解毒疗疮（尤擅清下焦湿热、退虚热） |
| **龙胆** | 苦，寒 | 胆、肝、胃经 | 清热燥湿，泻肝胆火（尤擅清肝胆湿热与实火） |
| **苦参** | 苦，寒 | 心、小肠、肝、肾、大肠经 | 清热燥湿，杀虫利尿，止痒（皮肤湿疹、阴痒要药） |

### 📍 清热凉血药
*清营血分实热，主治热入营血证（斑疹、吐衄、舌绛）。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **生地黄** | 甘、苦，寒 | 心、肝、肾经 | 清热凉血，养阴生津（凉血滋阴要药） |
| **玄参** | 甘、苦、咸，寒 | 肺、胃、肾经 | 清热凉血，滋阴降火，解毒散结 |
| **牡丹皮** | 苦、辛，微寒 | 心、肝、肾经 | 清热凉血，活血祛瘀（凉血而不留瘀） |
| **赤芍** | 苦，微寒 | 肝经 | 清热凉血，散瘀止痛（治血热瘀滞要药） |
| **紫草** | 甘、咸，寒 | 心、肝经 | 凉血活血，解毒透疹（透疹要药） |

### 📍 清热解毒药
*清热解毒，主治痈肿疮毒、温毒发斑、咽喉肿痛等各种热毒证。*

| :--- | :--- | :--- | :--- |
| **金银花** | 甘，寒 | 肺、心、胃经 | 清热解毒，疏散风热（疮家圣药，表里双清） |
| **连翘** | 苦，微寒 | 心、肺、小肠经 | 清热解毒，消肿散结，疏散风热（疮家圣药） |
| **蒲公英** | 苦、甘，寒 | 肝、胃经 | 清热解毒，消肿散结，利尿通淋（乳痈要药） |
| **紫花地丁** | 苦、辛，寒 | 心、肝经 | 清热解毒，凉血消肿（疔疮要药） |
| **大青叶** | 苦，寒 | 心、肺、胃、肝经 | 清热解毒，凉血消斑 |
| **板蓝根** | 苦，寒 | 心、胃经 | 清热解毒，凉血利咽（咽喉肿痛要药） |
| **穿心莲** | 苦，寒 | 心、肺、大肠、膀胱经 | 清热解毒，燥湿消肿 |
| **鱼腥草** | 辛，微寒 | 肺经 | 清热解毒，消痈排脓，利尿通淋（肺痈要药） |
| **土茯苓** | 甘、淡，平 | 肝、胃经 | 解毒，除湿，通利关节（治梅毒、湿疹关节痛） |
| **山豆根** | 苦，寒；有毒 | 肺、胃经 | 清热解毒，消肿利咽 |

### 📍 清虚热药
*清除虚热，主治骨蒸潮热、阴虚发热、热病后期伤阴虚热。*

| :--- | :--- | :--- | :--- |
| **青蒿** | 苦、辛，寒 | 肝、胆、肾经 | 清虚热，凉血，解暑，截疟（疟疾、暑热要药，不伤脾胃） |
| **地骨皮** | 甘，寒 | 肺、肝、肾经 | 凉血除蒸，清肺降火（清肺热骨蒸） |
| **白薇** | 苦、咸，寒 | 胃、肝、肾经 | 清热凉血，利尿通淋，解毒疗疮 |

## <span id="泻下药">泻下药</span>

> **分类概念**：引起腹泻或润滑大肠，促进排便的药物。主治胃肠积滞、便秘实热、水饮停聚等里实证。

### 📍 攻下药
*泻下作用峻猛，能通便泻火，多苦寒，适用于里实热证、大便秘结。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **大黄** | 苦，寒 | 脾、胃、大肠、肝、心包经 | 泻下攻积，清热泻火，凉血解毒，逐瘀通经（积滞便秘要药） |
| **芒硝** | 咸、苦，寒 | 胃、大肠经 | 泻下通便，润燥软坚，清火消肿（常与大黄相须为用） |
| **番泻叶** | 甘、苦，寒 | 大肠经 | 泻热行滞，通便 |

### 📍 润下药
*润燥滑肠，缓下通便。适用于年老、体虚、产后及津血不足之便秘。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **火麻仁** | 甘，平 | 脾、胃、大肠经 | 润肠通便（常用缓下药） |
| **郁李仁** | 辛、苦、甘，平 | 脾、大肠、小肠经 | 润肠通便，利水消肿 |
| **蜂蜜** | 甘，平 | 脾、肺、大肠经 | 补中缓急，润肺止咳，润肠通便，解毒 |

### 📍 峻下逐水药
*泻下作用极猛，能引起剧烈腹泻，排出体内水饮积聚。有毒，易伤正气，孕妇禁用。与甘草相反（十八反）。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **甘遂** | 苦，寒；有毒 | 肺、肾、大肠经 | 泻水逐饮，消肿散结（逐水力最猛，相反甘草） |
| **京大戟** | 苦、辛，寒；有毒 | 肺、肾、大肠经 | 泻水逐饮，消肿散结（相反甘草） |
| **芫花** | 苦、辛，温；有毒 | 肺、脾、肾经 | 泻水逐饮，祛痰止咳，杀虫疗疮（相反甘草） |
| **巴豆** | 辛，热；大毒 | 胃、大肠经 | 外用蚀疮，内服祛痰，峻下冷积（温下要药，大毒） |

## <span id="消导药（消食药）">消导药（消食药）</span>

> **分类概念**：以促进消化、消除宿食积滞为主要功效。多**甘温**，主入**脾、胃**经。

### 📍 消食化积药
*促进消化，主治宿食不消、脘腹胀痛、脾胃虚弱泄泻。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **山楂** | 酸、甘，微温 | 脾、胃、肝经 | 消食健胃，行气散瘀（尤擅消肉食油腻积滞） |
| **神曲** | 甘、辛，温 | 脾、胃经 | 消食健胃，和中止泻（尤擅消谷食金石，健脾和胃） |
| **麦芽** | 甘，平 | 脾、胃、肝经 | 消食健胃，回乳消胀，疏肝解郁（尤擅消米面谷食） |
| **谷芽** | 甘，温 | 脾、胃经 | 消食和中，健脾开胃 |
| **莱菔子** | 辛、甘，平 | 肺、脾、胃经 | 消食除胀，降气化痰（尤擅消食积气滞、咳喘） |
| **鸡内金** | 甘，平 | 脾、胃、小肠、膀胱经 | 消食健胃，涩精止遗，通淋化石（消食力强，兼化结石） |
| **槟榔** | 苦、辛，温 | 胃、大肠、脾经 | 杀虫消积，下气，利水（降气消积行水，治食积气滞） |

## <span id="止咳化痰平喘药">止咳化痰平喘药</span>

> **分类概念**：减轻或消除痰涎，制止咳嗽、气喘的药物。多**辛苦**，主归**肺**经。

### 📍 温化寒痰药
*性温燥，能温肺祛寒、燥湿化痰。主治寒痰、湿痰。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **半夏** | 辛，温；有毒 | 脾、胃、肺经 | 燥湿化痰，降逆止呕，消痞散结（湿痰要药，止呕要药） |
| **天南星** | 苦、辛，温；有毒 | 肺、肝、脾经 | 燥湿化痰，祛风止痉，散结消肿（风痰要药，治惊风中风） |
| **旋覆花** | 苦、辛、咸，温 | 肺、脾、胃、大肠经 | 消痰行水，降气止呕（降气化痰止呕） |
| **白前** | 辛、苦，微温 | 肺经 | 降气化痰（专于降肺气，温而不燥） |

### 📍 清化热痰药
*性寒凉，清热化痰。主治痰热郁肺之咳嗽咳痰。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **川贝母** | 苦、甘，微寒 | 肺、心经 | 清热润肺，化痰止咳，散结消痈（阴虚燥咳、肺燥要药，偏补） |
| **浙贝母** | 苦，寒 | 肺、心经 | 清热化痰，散结消痈（风热、痰热实证要药，偏清） |
| **瓜蒌** | 甘、微苦，寒 | 肺、胃、大肠经 | 清热涤痰，宽胸散结，润肠通便 |
| **竹茹** | 甘，微寒 | 肺、胃、心经 | 清热化痰，除烦止呕（清热止呕要药） |
| **桔梗** | 苦、辛，平 | 肺经 | 宣肺利咽，祛痰排脓（舟楫之剂，引药上行，入肺经） |
| **前胡** | 苦、辛，微寒 | 肺经 | 降气化痰，宣散风热 |

### 📍 止咳平喘药
*缓解或制止咳嗽喘息。适用于各类咳喘证。*

| 中药名 | 性味 | 归经 | 核心功效与考点 |
| :--- | :--- | :--- | :--- |
| **苦杏仁** | 苦，微温；有毒 | 肺、大肠经 | 降气止咳平喘，润肠通便（治喘要药，主含苦杏仁苷） |
| **紫苏子** | 辛，温 | 肺、大肠经 | 降气化痰，止咳平喘，润肠通便 |
| **紫菀** | 辛、苦，温 | 肺经 | 润肺下气，止咳化痰（温而不燥，治新久咳嗽） |
| **款冬花** | 辛、微苦，温 | 肺经 | 润肺下气，止咳化痰（常与紫菀相须为用） |
| **百部** | 甘、苦，微温 | 肺经 | 润肺下气，止咳，杀虫灭虱（肺痨久咳、百日咳要药） |
