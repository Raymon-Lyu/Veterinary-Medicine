# 外用方药
- 外用药以外用为主，通过涂敷、散布、喷洗、熏烟等形式治疗外科疾病的药物
- 以外用药为主要组成的方剂
### 常用中药
- 冰片
- 炉甘石
###### 麝香&冰片
**共性基础**：两药均为味辛、气香之品，走窜力强，均有开窍醒神之功。临床常相须为用，主治闭证神昏之证。

| **鉴别维度**  | **麝香**                                                         | **冰片**                    |
| --------- | -------------------------------------------------------------- | ------------------------- |
| **药性归属**  | 性温，气极香                                                         | 味苦，性微寒                    |
| **开窍走窜力** | **极强**（为醒神回苏之要药，走窜之力甚强）                                        | 较弱（开窍醒神与走窜之力均不及麝香）        |
| **闭证侧重**  | 宜治**寒闭**神昏（通过配伍亦可用治热闭）                                         | 更宜治**热闭**神昏               |
| **兼优功效**  | **活血通经、止痛、催产**                                                 | **清热解毒止痛**                |
| **特定主治**  | 1. 血瘀经闭、癥瘕、跌打损伤、痹证疼痛<br>2. 难产、死胎、胞衣不下等<br>3. 疮疡肿毒、咽喉肿痛（活血散肿止痛） | 火热目赤肿痛、喉痹、口疮、热毒疮疡肿痛及溃后不敛等 |
