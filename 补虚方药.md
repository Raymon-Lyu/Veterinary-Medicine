# 补虚方药
- 能够补益气血阴阳不足，治疗各种虚证
- 根据虚证类型，分为补气、补血、滋阴、壮阳
### 常用中药
##### 补气药
- 人参：大补元气，复脉固脱
- 党参：补中益气
- 黄芪
- 山药
- 白术
- 甘草
- 大枣
###### 人身vs 党参：药力等级与救脱能力
* **共性**：均能补脾益肺，生津养血。 
* **人参**（重症）：独具**大补元气、复脉固脱**之功，用于治气虚欲脱、脉微欲绝的危重证候。兼有益气安神、生血、摄血、壮阳作用。 
* **党参**（轻症）：功效类似但药力较弱，**无**大补元气、复脉固脱之功。**虽用大剂量，亦不能代替人参益气固脱**。 
 ###### 人参 vs 黄芪：内补脏腑与走表温升 
* **共性**：均为常用补气要药，同用可相互增强补气之功。 
* **人参**（主内）：能补心、脾、肺之气，长于大补元气、安神增智，为**治内伤气虚第一要药**。 
* **黄芪**（主表/升）：大补元气不如人参，但**温升之力强过人参**。长于**固表止汗、托毒生肌、利尿退肿**，善补肌表之气（此为人参所无或所不及）。 
###### 黄芪 vs 白术：升阳固表与健脾除湿 
* **共性**：两者均能补气、利水、止汗 
* **黄芪**（升提）：**脾、肺双补**，甘温补气力较强。长于**补气升阳**（适用于中气下陷、脱肛及脏气下垂）。主治气虚水肿，且**益卫固表、止汗之力优于白术**
* **白术**（除湿）：**仅补气健脾**。长于**健脾除湿**（主治脾虚失运、兼见水湿内停者）。治水湿内停时，即便湿盛而正虚不明显者亦可应用

###### 生地黄 vs 熟地黄
* **共性**：均能养阴滋阴。

 生地黄（性寒清热）熟地黄（微温滋补）
 
| **鉴别维度**  | **生地黄**                                           | **熟地黄**                                                  |
| --------- | ------------------------------------------------- | -------------------------------------------------------- |
| **性味归经**  | 甘苦，**性寒**                                         | 甘，**微温**                                                 |
| **核心功效**  | **清热凉血**，养阴生津                                     | **补血滋阴**，益精填髓                                            |
| **作用机制**  | 偏于**“清”与“凉”**（主治热证、火证）                            | 偏于**“补”与“填”**（主治一切精血虚损证）                                 |
| **典型主治**  | 1. 温病热入营血（舌绛烦渴）<br>2. 血热妄行（吐血、衄血、发斑）<br>3. 阴虚内热消渴 | 1. 血虚阴亏（心悸失眠、潮热骨蒸）<br>2. 精血不足（腰膝酸软、眩晕耳鸣、须发早白）<br>3. 阴亏消渴 |
| **核心排他性** | 是**清热凉血**的要药                                      | 经炮制后性转温，**绝对无清热凉血之功**                                    |

###### 天冬vs麦冬
两者药性均寒凉，养肺阴

| **鉴别维度** | **麦冬**         | **天冬**     |
| -------- | -------------- | ---------- |
| **寒润之力** | 较弱             | **强于麦冬**   |

| **作用部位** | **偏上**         | **偏下**     |

###### 杜仲vs续断
**共性基础**：两者均能补肝肾，强筋骨，止血安胎，疗伤续折。

|**鉴别维度**|**杜仲**|**续断**|
|---|---|---|
|**核心侧重**|以**补肝肾为重**|为补益肝肾，**宣通筋脉**之要药|
|**药性特点**|侧重于单纯的“补”|**补而不滞，形而不泄**|
|**优势场景**|治疗肝肾不足导致的腰膝酸痛、下肢痿软、胎动不安等纯虚证，以用杜仲为优|凡血脉郁滞不宣之证（如风湿痹痛，崩漏经多，跌扑损伤，骨折肿痛等），均可用续断|
|**特有功效**|可**降血压**|更强调兼顾活血与行滞|
