<center><h1>休克</h1></center>

## 概述
- 定义：**微循环**灌流量不足
### 分类
##### 心源性休克
- 心脏泵血能力不足 导致血液供应不足
- 心肌梗死、过速
##### 低血容量性休克
- 失血或脱水引起的血容量减少
- 失血性休克：消化道出血
- 失液性休克：呕吐、脱水、腹泻，心输出量降低
##### 血流分布异常性休克
- 过敏性休克：[[变态反应#过敏型超敏反应|过敏型超敏反应]]引起，如青霉素、血清制剂，血管活性物质如组胺等的释放引起血管扩张，有效循环血量减少
- 神经源性休克：受到过度刺激、惊吓
- 败血性休克：炎症因子大量释放，使得血液大量从血管内流出
### 结构基础

| 作用     | 血管          |
| -------- | ------------- |
| 阻力血管 | 微动脉/微静脉 |
| 分配血管 | 中动脉        |
| 容量血管 | 静脉          |

- 毛细血管前括约肌起到了“阀门”的作用，控制毛细血管网的灌注量
- 通常不存在毛细血管后括约肌，通过微静脉来进行“后闸门”的控制
微循环的三条通路：
1. 直截通路
2. 动-静脉短路
3. 迂回通路
### 微循环代谢及调节
微循环通过下列方式进行调节，即控制灌流量的变化
1. 机体代谢产物
	- 如$CO_2$、$H^+$
	- 代谢产物大多呈酸性会刺激毛细血管前括约肌开放引起血流灌注量增加([[#进行期]]变化的根本原因)
2. 神经调节
	- **调节过程**：交感-肾上腺髓质系统的的兴奋$\rightarrow$[[外周神经系统药理#儿茶酚胺类|儿茶酚胺]]分泌量增加$\to$血管收缩、动静脉短路开放
	- 这是[[#非进行期(non-progressive)|非进行期]]变化的根本原因，出现了皮肤苍白、四肢冰冷的原因
3. 体液调节
	- 缩血管物质：血管紧张素Ⅱ、抗利尿激素、血栓素A2、内皮素
	- 舒血管物质：组胺、缓激肽、TNF、NO、乳酸、腺苷
## 休克的过程
### 非进行期(non-progressive)
- **特点**：少灌少流，灌<流
- 机制：交感-肾上腺素髓质系统兴奋，导致儿茶酚胺类物质大量释放，通式伴有缩血管物质的释放，血管持续收缩，且前阻力血管收缩变化大于后阻力血管
	本质上是一种代偿机制，实现自我输血、自我输液，引起心输出量增加，血管外周阻力变大
- 临床变化从四个方面出发：
	- 皮肤与外观：面色苍白汗淋漓
	- CNS：烦躁不安
	- 泌尿系统：少尿
	- 心血管：脉压减小，血压甚至正常(收缩压可能保持不变，舒张压提高，脉压差减小)
### 进行期
- **特点**：多灌少流，灌>流，即微循环淤血
- 机制变化：
	- 毛细血管前括约肌对儿茶酚胺敏感性丧失$\to$血管扩张血液流入毛细血管网
	- 微静脉对酸中毒耐受性强，且在交感神经兴奋和组胺作用下持续收缩
	- 流体静压升高，大量血液积压在毛细血管内，造成**内皮损伤**
	- 血液变得**黏稠**，微循环阻力增加，但是总体血压相比非进行期仍降低
- 临床变化：
	- 心血管：**血压**呈现**进行性下降**。休克失代偿最典型的标志。回心血量不足导致心输出量下降，外周血管扩张导致阻力下降
	- **CNS**：**神志淡漠**甚至昏迷。因为此时血压太低，机体已经无法维持大脑的血液供应，脑缺血加重
	- **皮肤**：出现**发绀、花斑**。血液淤积在皮肤微血管里，氧气被耗尽，还原血红蛋白增多，所以皮肤呈现青紫色或大理石样的花纹
	- **肾脏**：**少尿或无尿**，肾血流量进一步减少，加上肾淤血，肾功能开始衰竭
### 不可逆期
- 微循环变化：
形成[[第三章 局部血液循环障碍#血栓类型|DIC]]
- 临床表现：
	- 血压未顽固性低血压，给予升压药也难以恢复
	- 浅表静脉塌陷严重
	- 皮下出现瘀斑、点状出血，区别于进行期皮肤的发绀、花斑
	- 出现MODS(多器官功能障碍综合征)
## 休克影响及救治
### 器官功能障碍
出现多器官功能障碍综合征，即短时间内同时或相继出现多个器官或系统的功能障碍或衰竭
##### 肺功能障碍
- 形成休克肺，肺出现充血、出血、水肿、DIC形成和肺泡内透明膜形成
##### 肾功能障碍
- 早期肾血流灌注量降低，肾小球滤过率降低，出现少尿
- 晚期肾小管直接堵塞了，少尿甚至无尿
##### 心功能障碍
- 心肌内出现DIC
- [[第八章 酸碱平衡紊乱#代谢性酸中毒|酸中毒]]伴发[[第七章 水和电解质代谢紊乱#高钾血症|高钾血症]]引起心肌收缩力下降
- 心脏本身的冠状动脉血流量降低，心功能被抑制
### 救治原则
按照VIPC原则来记忆：
- Ventilation：补充动脉血压分压，休克的最大危害是[[第六章 缺氧|缺氧]]
- Infusion：有效循环血量的扩容，先盐后糖，先晶后胶
- Pump：休克二期进行扩血管。休克早期使用缩血管物质，伴有强心药物
- Correction：纠正酸中毒，宁酸勿碱
总结如下：

| **治疗措施**     | **对应机制 / 目的**          | **关键注意事项**            |
| ------------ | ---------------------- | --------------------- |
| **1. 补充血容量** | 解决“有效循环血量$\downarrow$” | **首要措施**！需监测 CVP 和尿量。 |
| **2. 纠正酸中毒** | 恢复血管对药物的反应性            | 改善微循环淤血，防止高钾。         |
| **3. 扩血管药**  | **解除微静脉痉挛** (针对第二期)    | **前提是补足血容量**！         |
| **4. 缩血管药**  | 暂时维持心脑灌注压              | 仅用于早期或严重低血压，**不可滥用**。 |
| **5. 糖皮质激素** | 抗炎、抗毒、稳膜               | **早期、大剂量、短程**。        |
| **6. 肝素**    | 治疗 DIC (针对第三期)         | 仅用于 DIC **高凝期**。      |
