- Conception
	在致瘤因素下正常组织细胞基因发生了改变，增殖不受调控而形成的新生物，常形成肿块
## 肿瘤的特征
### 肿瘤的形态
肿瘤根据性质和对机体的影响可以分为**良性肿瘤**和**恶性肿**

|      | 良性肿瘤  | 恶性肿瘤   |
| ---- | ----- | ------ |
| 生长特性 | 通常有包膜 | 呈蔓延性生长 |
| 生长速度 | 慢     | 快      |
#### 息肉
- 突出于黏膜异常生长的组织，介于肿瘤和正常组织之间
### 肿瘤的一般结构
#### 实质
- 肿瘤的主要组成部分
可根据实质细胞形态追溯其来源
#### 间质
一般有结缔组织和血管构成，对肿瘤起支持和营养的作用
