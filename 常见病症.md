| 病证大类         | 证型               | 诊断特征（主证细节）                                                                                     | 治疗法则       | 涉及的方剂及中药加减                                                                                                                   |
| ------------ | ---------------- | ---------------------------------------------------------------------------------------------- | ---------- | ---------------------------------------------------------------------------------------------------------------------------- |
| **发热**(*外感*) | **1.外感风寒**       | 发热恶寒，恶寒重发热轻，无汗，皮紧毛乍，鼻流清涕，口色青白，舌苔薄白，脉浮紧，有时咳嗽，咳声洪亮。                                              | 辛温解表，疏风散寒。 | **麻黄汤**加减。<br>• 咳喘甚：加桔梗、冬花、紫菀。<br>• 兼表虚(恶风汗出脉浮缓)：用**桂枝汤**加减。<br>• 兼气血虚：用**发汗散**加减。<br>• 挟湿(恶寒发热/肢体沉重/少食纳呆/苔白腻)：用**荆防败毒散**加减。 |
|              | **2.外感风热**       | 发热重，微恶寒，耳鼻俱温，体温升高，或微汗，流黄/白色粘稠脓涕，咳嗽，咳声不爽，口干渴，舌稍红，苔薄白或薄黄，脉浮数；**牛鼻镜干燥，反刍减少**。                     | 辛凉解表，宣肺清热。 | **银翘散**加减。<br>• 热重：加黄芩、石膏、知母、花粉。<br>• 挟湿(体倦乏力/尿赤/黏膜黄染/苔黄腻)：去荆芥，加佩兰、厚朴、石菖蒲等。                                                  |
|              | **3.外感暑湿**       | 发热不甚或高热，汗出而身热不解，食欲不振，口渴，肢体倦怠沉重，运步不灵，尿黄赤，便溏，舌红，苔黄腻，脉濡数。                                         | 清暑化湿。      | **新加香薷饮**加味(香薷/厚朴/连翘/银花/鲜扁豆花/青蒿/鲜荷叶/西瓜皮)。<br>• 兼外感风寒又内伤饮食(食少呕呃/肠鸣泄泻)：用**藿香正气散**。                                             |
|              | **4.半表半里**       | 微热不退，寒热往来(发热恶寒交替)，脉弦。恶寒时精神沉郁、皮温低、耳鼻凉、腰拱毛乍、寒颤；发热时精神稍好、寒颤消失、皮温高、耳鼻转热。                            | 和解少阳。      | **小柴胡汤**加减。                                                                                                                  |
|              | **5.热在气分**       | 高热不退，但热不寒，出汗，口渴喜饮，头低耳耷，食欲废绝，呼吸喘促，粪干尿赤，口色赤红，舌苔黄燥，脉洪数。                                           | 清热生津。      | **白虎汤**加减。<br>• 热盛：加黄芩、黄连、银花、连翘。<br>• 伤津：加玄参、麦冬、生地。<br>• 尿短赤：加猪苓、泽泻、滑石、木通。                                                   |
|              | **6.热结胃肠**       | 高热，肠燥便干，粪球干小难下(或粪结不通/稀粪旁流)，腹痛，尿短赤，口津干燥，口色深红，舌苔黄厚而燥，脉沉实有力。                                      | 滋阴增液，清热泻下。 | **大承气汤** 或 **增液承气汤**加减。<br>• 高热：加银花、黄芩。<br>• 肚胀：加青皮、木香、香附等。                                                                  |
|              | **7.营分热**        | 高热不退，夜甚，躁动不安或神志昏迷，身上有出血点或出血斑，舌质红绛而干，脉细数。                                                       | 清营解毒，透热养阴。 | **清营汤**加减。                                                                                                                   |
|              | **8.血分热**        | 高热神昏，粘膜、皮肤发斑，尿血，便血，口色红绛，严重者抽搐。                                                                 | 清热凉血，熄风安神。 | **犀角地黄汤**加减。                                                                                                                 |
| **发热**(*内伤*) | **1.阴虚发热**       | 低热不退，午后更甚，严重者盗汗，少苔或无苔，唇干口燥，皮肤弹性降低，脉细数。                                                         | 滋阴清热。      | **青蒿鳖甲汤**<br>• 热重：加地骨皮、黄连、玄参等。<br>• 盗汗：加龙骨、牡蛎、浮小麦。<br>• 粪球干小：加当归(油炒)、肉苁蓉(油炒)等。<br>• 尿短赤：加泽泻、木通、猪苓等。                          |
|              | **2.气虚发热**       | 多于劳役后发热，耳鼻稍热，神疲乏力，易出汗，食欲减少，有时泄泻，舌质淡红，脉细弱。                                                      | 健脾益气，甘温除热。 | **补中益气汤**加减。                                                                                                                 |
|              | **3.血瘀发热**       | 外伤引起瘀血肿胀，局部疼痛，体表发热(或体温升高)；或产后瘀血未尽，伴腹痛及恶露不尽；口色红而带紫，脉弦数。                                         | 活血化瘀。      | • 外伤血瘀：用**桃红四物汤** 或 **血府逐瘀汤**。<br>• 产后血瘀：用**生化汤**加减。                                                                         |
| **泄泻**       | **1.寒泻**(*冷肠泄泻*) | 发病较急，泻粪稀薄如水(甚呈喷射状)，遇寒泻剧、遇暖泻缓，肠鸣如雷，食欲减少或不食，精神倦怠，头低耳耷，耳寒鼻冷，间有寒颤，尿清长，口色青白/青黄，苔薄白，口津滑利，脉沉迟。重者肛门失禁。 | 温中散寒，利水止泻。 | **猪苓散**加减。                                                                                                                   |
|              | **2.热泻**         | 发热，精神沉郁，食减或废绝，渴饮，有时轻微腹痛，蜷腰卧地；泻粪稀薄、粘腻腥臭，尿赤短，口色赤红，苔黄腻，口臭，脉沉数。                                    | 清热燥湿，利水止泻。 | **郁金散**加减。<br>• 热盛：去诃子，加银花、连翘。<br>• 水泻严重：去大黄，加车前子、茯苓、猪苓。<br>• 腹痛：加延胡索等。                                                      |
|              | **3.伤食泻**        | 食欲废绝(**牛羊反刍停止**)，肚腹胀满，隐痛；粪稀粘稠夹未消化食物，酸/恶臭，嗳气吐酸，放臭屁(或屁粪同泄)，**常伴呕吐(马属动物除外)**，泄吐后痛减；口色红，苔厚腻，脉滑数。   | 消积导滞，调和脾胃。 | **保和丸**加减。<br>• 食滞重：加大黄、枳实、槟榔。<br>• 水泻甚：加猪苓、木通、泽泻。<br>• 热盛：加黄芩、黄连。                                                           |
|              | **4.虚泻**(*脾虚泄泻*) | 形体羸瘦，毛焦肷吊，倦怠无力；病初食减，饮水增多，鼻寒耳冷，腹内肠鸣，粪中带水、渣粗或完谷不化；重者肛弛粪淌。舌淡白无苔，脉迟缓。后期四肢浮肿。                       | 补脾益气，利水止泻。 | **参苓白术散** 或 **补中益气汤**加减。                                                                                                     |
|              | **4.虚泻**(*肾虚泻*)  | 沉郁，头低耳耷，毛焦肷吊，腰胯无力，卧多立少，四肢厥逆，久泻不愈，**夜间及天寒时泻重**；重者肛门失禁，粪水外溢，腹下或后肢浮肿；口色如绵，脉沉细无力。                  | 温肾健脾，涩肠止泻。 | 1. **巴戟散**加减：去槟榔，加茯苓、猪苓等。<br>2. **四神丸** 合 **四君子汤**加减。                                                                        |
| **痢疾**       | **1.湿热痢**        | 精神短少，蜷腰卧地，食减或废绝，**反刍停止**，鼻镜干燥；弓腰努责，里急后重，泻粪不爽，下痢稀糊，赤白相杂或呈胶冻状(**湿重白多，热重血多**)；口色赤红，苔黄腻，脉数。        | 清热化湿，行气和血。 | 1. 选用**通肠芍药汤**加减。兼食滞者：加麦芽、神曲等。<br>2. 马、犬、猫、猪等可用**白头翁汤**加减。                                                                   |
|              | **2.虚寒痢**        | 精神倦怠，毛焦体瘦，鼻寒耳冷，四肢发凉，食欲及反刍日减；不时努责，泻痢不止，水谷并下，带灰白或泡沫状，时有腹痛；重者失禁或带血；口色淡白/灰白，苔白滑，脉迟细。               | 温脾补肾，收涩固脱。 | **四神丸** 合 **参苓白术散**加减。<br>• 寒甚：加肉桂。<br>• 腹痛明显：加木香。<br>• 久痢不止：加诃子。<br>• 便中带血：加血余炭、炒地榆。<br>• 里急后重：加枳壳、青皮。                      |
|              | **3.疫毒痢**        | 发病急骤，高热烦躁，食减或废绝；弓腰努责，里急后重，腹痛起卧，泻粪粘腻夹脓血，腥臭难闻；口色赤红，苔干黄，脉洪数或滑数。                                   | 清热燥湿，凉血解毒。 | **白头翁汤**加减。<br>• 热毒甚：加马齿苋、银花、连翘。<br>• 腹痛明显：加白芍、甘草。<br>• 渴饮贪饮：加葛根、麦门冬、玄参、沙参。<br>• 里急后重剧烈：加枳壳、槟榔。                              |
