
| 病证大类       | 证型               | 诊断特征（主证）                     | 治疗法则       | 涉及的方剂及中药加减                                                                                                    |
| ---------- | ---------------- | ---------------------------- | ---------- | ------------------------------------------------------------------------------------------------------------- |
| **发热**(外感) | **外感风寒**         | 恶寒重，发热轻，无汗，鼻流清涕，皮紧毛乍，脉浮紧。    | 辛温解表，疏风散寒。 | **麻黄汤**加减。<br>• 咳喘甚：加桔梗、冬花、紫菀。<br>• 兼表虚（汗出/脉浮缓）：用**桂枝汤**加减。<br>• 兼气血虚：用**发汗散**加减。<br>• 挟湿（肢体沉重）：用**荆防败毒散**加减。 |
|            | **外感风热**         | 发热重，微恶寒，微汗，流黄/白稠涕，口渴，脉浮数。    | 辛凉解表，宣肺清热。 | **银翘散**加减。<br>• 热重：加黄芩、石膏、知母、花粉。<br>• 挟湿（体倦/尿赤/苔黄腻）：去荆芥，加佩兰、厚朴、石菖蒲等。                                          |
|            | **外感暑湿**         | 发热不甚或高热，汗出而身热不解，倦怠沉重，便溏。     | 清暑化湿。      | **新加香薷饮**加味（香薷、厚朴、连翘、金银花、鲜扁豆花、青蒿、鲜荷叶、西瓜皮）。<br>• 兼内伤饮食（呕呃/泄泻）：用**藿香正气散**。                                      |
|            | **半表半里**         | 微热不退，寒热往来（恶寒与发热交替出现），脉弦。     | 和解少阳。      | **小柴胡汤**加减。                                                                                                   |
|            | **热在气分**         | 高热不退，但热不寒，口渴喜饮，呼吸喘促，粪干尿赤。    | 清热生津。      | **白虎汤**加减。<br>• 热盛：加黄芩、黄连、银花、连翘。<br>• 伤津：加玄参、麦冬、生地。<br>• 尿短赤：加猪苓、泽泻、滑石、木通。                                    |
|            | **热结胃肠**         | 高热，肠燥便干，粪球干小难下或稀粪旁流，脉沉实有力。   | 滋阴增液，清热泻下。 | **大承气汤**或**增液承气汤**加减。<br>• 高热：加银花、黄芩。<br>• 肚胀：加青皮、木香、香附等。                                                     |
| **发热**(内伤) | **阴虚发热**         | 低热不退，午后更甚，严重者盗汗，少苔或无苔。       | 滋阴清热。      | _(资料未提主方名，但记载了加减)_<br>• 热重：加地骨皮、黄连、玄参等。<br>• 盗汗：加龙骨、牡蛎、浮小麦。<br>• 粪球干小：加当归(油炒)、肉苁蓉(油炒)。<br>• 尿短赤：加泽泻、木通、猪苓等。   |
|            | **气虚发热**         | 多于劳役后发热，神疲乏力，易出汗，食欲减少，脉细弱。   | 健脾益气，甘温除热。 | **补中益气汤**加减。                                                                                                  |
|            | **血瘀发热**         | 外伤致局部肿胀疼痛、体表发热；或产后恶露不尽伴腹痛。   | 活血化瘀。      | • 外伤血瘀：用**桃红四物汤**或**血府逐瘀汤**。<br>• 产后血瘀：用**生化汤**加减。                                                            |
| **泄泻**     | **寒泻****(冷肠泄泻)** | 发病较急，泻粪稀薄如水，遇寒泻剧、遇暖泻缓，肠鸣如雷。  | 温中散寒，利水止泻。 | **猪苓散**加减。                                                                                                    |
|            | **热泻**           | 发热，渴饮，泻粪稀薄、粘腻腥臭，尿赤短，苔黄腻。     | 清热燥湿，利水止泻。 | **郁金散**加减。<br>• 热盛：去诃子，加银花、连翘。<br>• 水泻严重：去大黄，加车前子、茯苓、猪苓。<br>• 腹痛：加延胡索等。                                       |
|            | **伤食泻**          | 食欲废绝，肚腹胀满，粪稀粘稠且夹有未消化食物，酸臭。   | 消积导滞，调和脾胃。 | _(资料中未列出具体代表方，通常依治则配用消导药)_。                                                                                   |
| **痢疾**     | **虚寒痢**          | 精神倦怠，四肢发凉，泻痢不止，水谷并下，带灰白/泡沫状。 | 温脾补肾，收涩固脱。 | **四神丸**合**参苓白术散**加减。<br>• 寒甚：加肉桂。<br>• 腹痛：加木香。<br>• 久痢不止：加诃子。<br>• 便中带血：加血余炭、炒地榆。<br>• 里急后重：加枳壳、青皮。           |
|            | **疫毒痢**          | 发病急骤，高热烦躁，弓腰努责，泻粪粘腻夹脓血，腥臭难闻。 | 清热燥湿，凉血解毒。 | **白头翁汤**加减。<br>• 热毒甚：加马齿苋、银花、连翘。<br>• 腹痛：加白芍、甘草。<br>• 渴饮：加葛根、麦门冬、玄参、沙参。<br>• 里急后重剧烈：加枳壳、槟榔。                   |
