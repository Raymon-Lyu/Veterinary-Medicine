# 旺旺病例讨论
> [!info] 基本信息
> - **姓名**：旺旺
> - **品种**：中华田园犬
> - **年龄**：5岁
> - **性别**：雄性，已绝育
> - **主诉**：慢性皮肤病灶

## 病史追问
1. **皮肤问题最早从什么时候开始，最先出现在哪个部位？**
	 - 目的：判断起病方式、首发部位及扩散顺序。
2. **是否瘙痒？抓挠、舔舐、摩擦的程度如何？**
	- 目的：区分瘙痒性疾病与以脱毛/皮脂异常为主的疾病。
3. **既往是否做过治疗？用了哪些药？效果怎样？**
	- 重点询问：抗生素、激素、驱虫药、药浴。
4. **平时是否规律进行体外/体内驱虫？最近一次是什么时候？**
	- 目的：排查跳蚤、疥螨、蠕形螨等寄生虫相关疾病。
5. **近期有无全身症状？**
	- 如：精神食欲下降、体重变化、多饮多尿、反复感染等。
	- 目的：筛查潜在基础病或免疫异常。

## 病变部位总结

根据病例图片，病灶主要累及以下部位：

### 1. 颈侧 / 肩前区
- 大片脱毛
- 色素沉着明显
- 皮肤增厚、粗糙
- 提示慢性炎症后改变

### 2. 体侧 / 背腰部
- 多发圆形或类圆形斑块状病灶
- 脱毛明显
- 皮肤发黑、表面粗糙
- 可伴鳞屑、皮脂增多

### 3. 胸前 / 腋下附近
- 较大局灶性斑块
- 局部结痂、糜烂样改变
- 周围有红斑和色素沉着
- 提示继发感染或自体损伤

### 4. 腹侧 / 腹股沟或内侧皮肤
- 弥漫性发红
- 轻度脱毛
- 皮肤炎症表现较明显

### 病灶整体特征概括
- **慢性**
- **泛发性 / 多灶性**
- **以脱毛、色素沉着、皮肤增厚为主**
- **局部伴红斑、结痂、继发感染表现**

## 选择化验检查
### 1. 皮肤刮片
- 首选检查
- 目的：排查 **蠕形螨、疥螨等寄生虫性皮肤病**
- 对本病例最关键

### 2. 胶带取样（皮肤细胞学）
- 判断是否存在：
  - 细菌（球菌/杆菌）
  - 马拉色菌
  - 炎性细胞
- 用于指导是否需要抗菌/抗酵母治疗

### 3. 血液学检查
- 评估炎症程度
- 排查成年犬泛发性皮肤病背后的全身问题

### 4. 尿液分析
- 作为低成本全身筛查补充
- 有助于评估潜在基础病

### 5. 真菌培养
- 虽非首位怀疑，但多发脱毛斑仍需与皮肤癣菌病鉴别

## 初步诊断思路

### 最可能诊断
- **泛发性蠕形螨病**
- **继发性细菌性皮肤炎 / 浅表或深部脓皮症**
- 伴：
  - 慢性皮脂溢
  - 色素沉着
  - 苔藓样变

### 鉴别诊断
1. 皮肤癣菌病
2. 过敏性皮肤病并继发感染
3. 内分泌性脱毛伴继发感染
4. 疥螨感染
5. 其他角化异常性皮肤病

## 治疗原则

### 1. 病因治疗：驱螨
- 以有效的抗蠕形螨方案为核心
- 足疗程治疗
- 定期复查皮肤刮片
- 连续多次阴性后再考虑停药

### 2. 控制继发细菌感染
- 若细胞学提示细菌感染，应进行抗菌治疗
- 轻症可局部治疗
- 中重度或泛发性可加系统抗菌药
- 反复发作或治疗失败时建议做细菌培养+药敏

### 3. 局部皮肤护理
- 药浴
- 清理痂皮与鳞屑
- 降低皮肤表面微生物负荷
- 改善皮脂溢和异味

### 4. 谨慎使用抗炎药
- 在未排除寄生虫病前，不宜贸然长期使用糖皮质激素
- 避免加重螨虫增殖和感染扩散

### 5. 筛查并处理基础病
- 若为成年起病泛发性蠕形螨病，应进一步评估：
  - 内分泌异常
  - 慢性消耗性疾病
  - 免疫状态异常
  - 既往不当用药史

### 6. 复查
- 复查病灶面积、红斑、结痂、瘙痒程度
- 重复刮片与细胞学检查
- 根据结果调整疗程

## 列举用药
> 以下药物为“按该病例最可能诊断路径可考虑的常见用药类别”，具体是否使用、剂量和疗程应依据体重、合并症和检查结果决定

### （一）驱螨药
1. **氟雷拉纳**：异噁唑啉类，用于螨虫/跳蚤/蜱虫控制
2. **阿福拉纳**：异噁唑啉类，常用于犬外寄生虫控制，也常见于蠕形螨治疗方案中
3. **沙罗拉纳**：异噁唑啉类，用于犬跳蚤蜱虫控制，也常用于蠕形螨相关病例管理

### （二）继发细菌感染控制
1. **氯己定洗剂 / 香波**：用于局部抗菌、降低皮肤细菌负荷
2. **氯己定 + 咪康唑复方香波**：适合伴细菌/酵母菌过度生长时局部管理
3. **系统性抗菌药**：根据细胞学结果选择合适抗生素，必要时进行药敏培养指导用药

### （三）皮脂溢和角化异常护理
1. **抗菌/控脂药浴**
2. **保湿修复类皮肤护理产品**
3. **必要时配合去角质护理**
